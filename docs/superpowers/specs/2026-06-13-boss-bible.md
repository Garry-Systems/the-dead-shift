# Boss Bible — Survivor Game (Ryan Ace)

_Living catalog of 58 boss concepts. Spectacle-tier, anything-goes, designed around the no-aim / shoot-while-still / dash combat model._

## How to use this bible

This is a living catalog, not a finished list. Bosses are added over time by composing the reusable template below — pick a family, a craziness tier, and a small stack of attack patterns, then slot them into a phase table. When a new "what IS that" idea lands, write it as a one-liner, decide its build-cost tier, and append it as its own entry; the front matter (this section through Overlap notes) is the only part you maintain by hand as the roster grows.

## The reusable Boss template

Every boss is one `BossBase` node + a list of `AttackPattern` objects + a phase table. Adding a boss should mean writing one new pattern or two and a phase table — not a new engine.

### `BossBase` (extends the existing `Boss.gd`)
Already provides: `enemies`+`boss` group membership, `take_damage`, `ignite`, chase-the-player, `health_fraction()`, on-death XP burst + full-heal + relic drop, bottom-center HP bar. Add to it:
- `phases: Array` — ordered list of `{ at_fraction: float, patterns: Array, move_profile, on_enter: Callable }`. `_process` watches `health_fraction()` and fires `on_enter` when it crosses a threshold.
- `active_patterns: Array[AttackPattern]` — swapped wholesale on phase change.
- `pattern_clock` — a single `Timer` (or accumulator) that asks the current phase for the next pattern to run (weighted pick or round-robin).
- Optional hooks a phase can set: `invulnerable: bool`, `damage_mult`, `target_override` (for taunt/decoy bosses), `move_speed_mult`.

### `AttackPattern` interface
A tiny script/resource with three beats — this IS the SlamWave loop generalized:

```
telegraph(boss, player) -> draws/places the warning, returns a duration
execute(boss, player)   -> the actual hit (spawn band/projectiles/zone)
# the gap between them is the dodge window; dash is always the out
```

A pattern owns its own `_draw` telegraph and cleans itself up. The boss just calls `telegraph()`, waits the returned time, calls `execute()`. Readability rule baked in: telegraph duration is clamped to 0.5–1.2s and every telegraph is high-contrast.

### Library of reusable telegraph primitives
Build these once; every boss reuses them.
- **ExpandingRing** — SlamWave verbatim: faint fill → drawn ring → leading band aimed at player. Dash through it.
- **AimedBand** — straight line/wedge snapped at the player's position; the "laser/charge" primitive.
- **ZoneFill** — paint a region (circle/rect/tile-set) that becomes a damaging `Area2D` after the warning. Backbone of acid/fire/void grids.
- **ProjectileEmitter** — spawn N `Bullet`s in a fan / ring / spiral / aimed shot; param: count, arc, speed, spin.
- **SummonSpawner** — instance `Enemy.gd` variants (adds, eggs, decoys); optional `decoy_priority` tag to steal auto-aim.
- **DebuffApplier** — flip a flag on Player/Gun for a clamped window: `hold_fire`, `move_speed_mult`, `dash_locked`, axis-remap. Always paired with a screen cue.

### How a designer builds a boss
1. Pick a family + craziness tier + the one-liner.
2. Choose 2–4 patterns from the library (or write one new one — that's the "new system" column).
3. Write the phase table: e.g. `[{100%: [Ring, Fan]}, {60%: [Ring, Fan, Summon], faster}, {25%: [enrage: Spiral+Band, invuln-pulse]}]`.
4. If it needs a genuinely new system (tile grid, input override, shader overlay), build it as one self-contained node and expose a single flag the phase table flips.

That's the whole contract: `BossBase` + a phase table + patterns from the shelf. Anything that doesn't fit becomes exactly one new flagged system.

## Wave-escalation ladder

Bosses spawn every 5 waves. Early bosses teach one mechanic cleanly; later ones stack systems, override controls, and escalate spectacle. Tier = the `crazy` rating; readability stays high at every depth.

| Wave | Craziness band | Families to draw from | Example bosses (from the set) |
|---|---|---|---|
| **5** (first boss) | 4, single clean mechanic | Defensive/Attrition, Speed/Duel, simple Arena Control | Aegis Tortoise, Vel the Quickening, The Brood Mother, The Metronome |
| **10** | 4, one new system | Swarm/Summoner, Elemental, Conditional Vulnerability | Cleaver the Halving, Cindra, Hoarvault, The Lighthouse, The Lure-Glutton |
| **15** | 4–5, layered hazard / decoy | Arena Control, Multi-Phase, Auto-aim corruption | Conveyor Saint, Heart of the Maze, Carousel of Saints, GEMINI ENGINE, Volkarr |
| **20** | 5, debuff that disables the player | Anti-Player Systems, Darkness/Audio, Build-Hijack | STATIC, OVERCLOX, The Long Dark, The Silent Movie, The Pickpocket |
| **25** | 5, control-override / reality-bend | Surreal/Reality-Bending, Input override | The Other Ryan, The Migraine, MIRRORMAW, PUPPETINE, The Inversion, Chronos |
| **30+** (endgame / set-piece) | 5, screen-as-boss & living arena | Environmental Set-Piece, the biggest spectacles | Atlas Unbound, The Glitch, The Moother of the Border, The Auteur, Mephistopheles, The Maestro of Last Breaths |

Cycle within a band on repeat clears (wave 35, 40…) and reshuffle so the same boss doesn't recur back-to-back.

## Flagship 10

The ten must-build bosses — chosen for variety of family, maximum "what IS that" factor, and clean readability despite the craziness.

1. **The Tide-Glutton** (Arena Control) — the rising acid sea on shrinking pixel-islands is the single most legible "the floor is the enemy" fantasy and directly weaponizes shoot-while-still.
2. **Carousel of Saints / The Orrery** (Decoy Orbit) — the auto-aim locking onto the WRONG moon turns the game's own targeting into the boss; pure mechanic, pure spectacle.
3. **The Migraine** (Input Debuff) — a telegraphed bullet-ring where the gaps are real but your inputs mirror; the cleanest, safest control-override on the roster.
4. **The Chorus of One / Mirrorman** (Movement Echo) — a clone replaying YOUR last seconds that steals auto-aim; uncanny, screenshot-worthy, and one small ring-buffer.
5. **OVERCLOX, the Heat Tyrant** (Anti-Player) — turning the player's own auto-fire into an overheating bomb inverts the core loop with one meter; brilliant tension, cheap to build.
6. **The Long Dark** (Vision-Denial) — total black arena lit only by your own muzzle flash is an instant marketing screenshot and a self-contained overlay.
7. **The Inversion** (Multi-Phase Transformer) — rewrites one control rule per phase (fire-while-moving, reverse dash, farthest-target auto-aim); a masterclass in escalating dread from one tiny RuleState.
8. **Mr. Mirror** (Reflect-its-own-attack) — bullet-immune; killable ONLY by dash-parrying its homing reflection into its face. A whole boss from one verb the player already owns.
9. **Atlas Unbound** (Living Arena) — the arena walls rip up and fold into a titan made of your floor; the ultimate endgame set-piece and the "there's no boss on the field… the FIELD is the boss" reveal.
10. **Mephistopheles, the Quiet Investor** (Economy/Greed) — a devil that never attacks, only offers irresistible deals whose fine print edits your combat model; a thinking-person's boss unlike anything else in the set.

## Overlap notes

Several bosses share a mechanical core. Build ONE excellent version of each cluster, then differentiate the rest by theme/family or cut them.

- **Movement ring-buffer echo (4 bosses):** The Chorus of One, Mirrorman, The Other Ryan, and The Maestro of Last Breaths all replay recorded player movement as auto-aim-stealing clones. Build the recorder ONCE. Differentiate: *The Other Ryan* = 1 delayed self (Surreal duel), *Mirrorman* = rushdown + decoys (Speed family), *The Maestro* = records the WHOLE arena including zombies and bullets (the maximal version). The Chorus of One is the weakest — fold it into Mirrorman or recolor as a Swarm variant.

- **Input-inversion / control override (5 bosses):** The Migraine, MIRRORMAW, The Inversion, PUPPETINE, and The Glitch all override player input. Keep *The Migraine* (axis mirror, the gentle teaching version), *The Inversion* (per-phase rule rewrites, the deep version), and *PUPPETINE* (full possession, the scariest). MIRRORMAW's "drag left = go right + must-move-to-shoot" is a subset of The Inversion's phases — merge it in as one Inversion phase, or keep only if its two-faced theme justifies a standalone.

- **Auto-aim decoy / ghost bait (5 bosses):** Carousel of Saints, STATIC, Telephage, Polyphemus Static, and Volkarr's rod all spawn fake targets that steal nearest-target auto-aim. Carousel (orbiting moons) and STATIC (TV ghosts) are the two distinct flavors worth building; Telephage and Polyphemus Static are nearly identical (static-wraith + decoys + nearest-only vulnerability + screen shader) — **merge them into one boss.**

- **Acid/fire/void floor grid (4 bosses):** The Tide-Glutton (acid), Cindra (fire), The Cartographer (void), and The Lure-Glutton (acid pools) all use a tile/zone hazard grid. They read differently enough to keep, but build them on ONE shared `ZoneFill`/coarse-cell-grid system. The Lure-Glutton's pools are just a 3-cell static case of the same code.

- **Heal-link / invuln-until-adds-die (2 bosses):** Choirmother (larvae feed HP) and the boss-shield variants in The Brood Mother / Null Verse overlap on "kill the support to expose the boss." Choirmother is the definitive version; keep Brood Mother distinct by making its shield about screen-crowding (auto-aim can't *reach* her) rather than a heal tether.

- **Time manipulation (2 bosses):** Chronos (slows player-local time, rewinds player) and The Maestro (records/replays the arena) both bend time. Different enough — Chronos is a debuff duel, Maestro is a ghost set-piece — but don't build a third time boss.

- **Gravity/drift pull (3 bosses):** Conveyor Saint (rotating conveyor), Vertigo (gravity tilt), and Null-Maw (radial pull) all add a global velocity vector to the player. Same `DirectionalDrift` primitive, three directions (rotating / constant-downhill / radial-inward). Build the primitive once; the three are cheap reskins and all worth keeping for variety.

# The Bosses (58)

## Family: Surreal / Reality-Bending (5)

### The Other Ryan — a mirror-glitch doppelganger of YOU that copies your movement on a delay and turns your own dashes into death sentences
- **Spectacle:** A second Ryan flickers into existence with the exact same hooded-survivor sprite, but rendered with an inverted/cyan palette (swap `flash_color`/tint, add a chromatic-split jitter every ~0.4s). It moves like a ghost echo of you, 0.6s behind — when you strafe left, a heartbeat later HE strafes left. On a phone this is instantly legible: there are now TWO of the same character on screen, one of them wrong-colored and laggy. The first time players see their own dash get mirrored back at them as an attack, that's the screenshot.
- **Signature mechanic:** **Movement mimicry via a delayed input replay.** A small ring-buffer records the player's per-frame velocity; the boss plays it back N frames late, so the doppelganger literally retraces the player's path. Crucially: when the RECORDED frame was a dash, the replay frame becomes a damaging lunge with a brief leading hitbox — your own dodge, weaponized against you a second later.
- **Attack patterns:**
  - **Echo Lunge:** telegraph = the doppelganger's body flashes white and a thin cyan arrow streak draws along the path you dashed ~0.6s ago -> execution = he dashes that same vector at 2x speed with a hit hitbox on his body -> dodge: because it follows YOUR old path, you can pre-bait it (dash toward a wall/corner, then sidestep so his copied lunge sails into empty space). Dash to break contact (note: dashing again just feeds the buffer another lunge in 0.6s, so dash with intent).
  - **Sync Slam:** telegraph = both Ryans freeze and a shared faint filled circle (reuse SlamWave windup) blooms under BOTH of you simultaneously -> execution = two expanding rings, one centered on him, one centered on where you were at telegraph start -> dodge: dash perpendicular out of both bands. Standing still to shoot here is lethal because one ring is centered on your last position.
  - **Palette Tear (phase 2+):** telegraph = screen edges glitch with cyan scanlines for ~0.8s -> execution = the doppelganger briefly becomes the NEAREST enemy, stealing your auto-aim onto himself -> counter: keep firing into him (he's a valid target), but it means your bullets are committed to him while trash closes in.
- **Exploits the combat model:** It is the purest possible attack on *shoot-only-while-still*. To deal DPS you must stand still — but standing still makes your movement trail predictable and short, so the doppelganger's replayed lunges converge right where you're rooted. The boss literally punishes the exact stillness your gun requires. And because dash is your only out, every dodge you make is *recorded and returned*, so the player must learn to dodge in directions that are safe to be copied. It also abuses auto-aim-nearest: the doppelganger shares your sprite, so in a crowd it's visually hard to confirm your gun is locked onto HIM vs a trash mob beside him.
- **Phases:** 100-66%: 0.6s replay delay, lunges only. 66-33%: delay tightens to 0.4s (echo feels closer/scarier), Sync Slam added. <33%: spawns a SECOND fainter doppelganger replaying at a different delay (0.9s) so two echoes crisscross — plus the Palette Tear auto-aim steal goes on a short loop.
- **Counterplay:** A skilled thumb learns to 'lead' the boss into bad ground: dash toward open space, immediately reposition, and dump fire while the copied lunge wastes itself. The lesson is intentionality — you can no longer dash reflexively, because the boss gives your panic-dashes back to you with teeth. Players who internalize 'dash on purpose, stand still on safe ground' beat it clean.
- **Difficulty knobs + wave depth:** Knobs = replay delay (frames), lunge hitbox lifetime, lunge speed multiplier, second-echo HP threshold, Palette-Tear cooldown. Debut at **wave 15** (players need a dash habit to subvert). Lower delay + faster lunge for wave 25/30 reprises.
- **Reward twist:** Drops a unique relic **'Cracked Mirror'** — short window of post-dash i-frames (turns your dash trail safer), thematically the boss's own trick handed to you.
- **Godot build notes:** Reuse `Boss.gd` (groups/health/flash/reward) for the chassis and the existing Player `DIR_TEX` sprite set tinted cyan via the flash shader's `base_tint`. Reuse `SlamWave.gd` verbatim for Sync Slam (two instances). NEW small system: a fixed-size `PackedVector2Array` ring-buffer on the boss that samples `player.velocity` (and a parallel bool buffer for 'was dashing') every `_physics_process`; index back by `delay_frames` to drive the doppelganger's `velocity`. Echo Lunge = when the replayed bool is true, multiply velocity and enable a body Area2D hitbox for ~0.15s. Palette Tear = temporarily nudge his effective distance so auto-aim (nearest) snaps to him. No camera/screen shader strictly required (chromatic jitter is optional polish via a tiny canvas_item shader).

### The Bullet Choir — a hollow silhouette that EATS your bullets and rebuilds its body out of them, then fires your own ammo back as a wall
- **Spectacle:** It enters as a translucent, hollow chalk-outline of a humanoid — you can see the arena through it. As you fire, each bullet flies INTO the outline and lodges there, glowing; the silhouette fills in, bullet by bullet, until it's a solid figure made entirely of your own glowing ammo. Then it inhales and exhales the whole thing back as a fan of bullets. On a phone the read is unmistakable: a body literally assembling from your shots, brighter the more you fire. Pure 'what IS that.'
- **Signature mechanic:** **Bullet absorption and reflection.** It does not take normal damage while 'hollow' — instead it CONSUMES your bullets to fill a charge meter (visualized as how 'solid' it looks), then converts that exact stored count into a return volley aimed back. Your DPS becomes its ammo. It's the only boss whose threat is fed directly by your own firing.
- **Attack patterns:**
  - **Drink (passive/ongoing):** no telegraph needed — while hollow, your bullets curve slightly toward it and vanish into the outline, charging it. The visual fill-level IS the warning. Counter: STOP shooting it (move, breaking your auto-fire) to deny charge — but auto-aim wants to keep feeding it.
  - **Choir Volley:** telegraph = the now-solid figure flashes bright and a fan of faint aim-lines sweeps to show the spread arc for ~0.9s -> execution = it fires back roughly as many bullets as it absorbed, in a fan (more absorbed = denser wall) -> dodge: dash through a gap in the fan; the denser it is (the more you fed it), the tighter the gap, so over-feeding directly raises the dodge difficulty.
  - **Solid Window:** telegraph = right AFTER a volley the figure goes hollow→briefly SOLID and dark for ~1.5s (clearly different sprite/color) -> execution = during this window it IS damageable normally and does NOT absorb -> counter: this is your real DPS window; stand still and unload, then move before it goes hollow again.
  - **Crescendo (phase 3):** telegraph = it pulls trash enemies into itself, each adding a 'voice', and the outline pulses -> execution = a 360-degree ring volley instead of a fan -> dodge: pre-position near the boss in a pre-spotted gap and dash radially out.
- **Exploits the combat model:** This is a frontal assault on **auto-aim-nearest** and the player's total lack of a fire toggle. The player cannot choose NOT to shoot — the gun auto-fires at the nearest target whenever you're still, and if the Choir is nearest, you are *forced* to feed it. The only ways to stop charging it are to keep moving (no fire) or to put a trash enemy between you and it to steal the auto-aim — but then you're not killing the boss either. It turns the game's auto-fire convenience into a liability and makes 'when am I allowed to shoot the boss' the entire puzzle (answer: only in the Solid Window).
- **Phases:** 100-66%: short hollow phases, small volleys, generous Solid Windows. 66-33%: hollow phases longer, volleys denser, Solid Window shrinks. <33%: Crescendo 360 volleys + it absorbs faster, so greedy players who never stopped feeding it now face near-unwalkable walls — punishing exactly the players who ignored the deny-charge lesson.
- **Counterplay:** The skilled line is *fire discipline without a fire button*: kite to deny charge, body-block auto-aim with trash when needed, and burst only during Solid Windows. Players learn that in this game 'always be shooting' is not always correct — sometimes movement is the right play even though it costs DPS.
- **Difficulty knobs + wave depth:** Knobs = absorb-to-volley ratio, volley fan width/gap size, Solid Window duration, hollow-phase length, Crescendo threshold. Debut **wave 25** (needs players who've built DPS and must now restrain it). Its effective HP auto-scales with player DPS, so it's naturally self-balancing across builds.
- **Reward twist:** Drops **'Echo Rounds' relic** — a small % of YOUR bullets that miss come back as a free second shot (you become a mini version of it).
- **Godot build notes:** Reuse `Boss.gd` and `Bullet.gd`. NEW small system: an `Area2D` 'mouth' on the boss in a high-priority absorb mode — when a `Bullet` enters and the boss is hollow, `queue_free()` the bullet, increment `_absorbed`, and update a `_draw()`/material `fill` uniform (lerp the silhouette alpha by `_absorbed/cap`). Choir Volley spawns `_absorbed` Bullet instances in a fan via `lerpf` across an arc (the game already fans shotgun pellets exactly this way). Solid Window = flip a `_solid` bool that routes hits to normal `take_damage` and disables absorb. Curve-toward-boss on player bullets is optional polish (steer bullet velocity slightly toward the boss each frame while hollow).

### The Auteur, Director of the Last Reel — A film-projector god who turns the arena into a movie set, slaps a glowing 'ACTION' box on the floor, and forces you to perform your fight inside its frame or be cut.
- **Spectacle:** A giant pixel film projector hovers at the top of the arena, beam of dusty light fanning down. A bright white director's FRAME (a rectangle with cinema corner-ticks) is projected onto the floor and pans around like a roaming camera shot. Black letterbox 'film bars' slam in from the top and bottom of the SCREEN itself during cutaways. On a phone this reads instantly: a glowing box = 'stand here', black bars = 'something's about to happen'. Pixel film-grain particles drift down the beam. When it kills you it 'cuts to black' with a hand-lettered THE END card.
- **Signature mechanic:** THE FRAME. There is always exactly one projected camera frame on the floor. Anything OUTSIDE the frame is 'off-camera' — your bullets that leave the frame fizzle into film-grain (capped range inside the box), and crucially the boss is ONLY damageable while it is inside the frame. The frame pans and shrinks on its own schedule, dragging the 'valid combat zone' away from you, so you must keep repositioning into the shot to keep your DPS legal — and repositioning means moving, which means not shooting.
- **Attack patterns:**
  - 'ACTION!' (zone shove): telegraph -> the frame flashes yellow and slides to a new spot over ~1.0s with a dotted path line -> execution: the old frame edge becomes a hard 'CUT' line that sweeps across, dealing a hit to anyone it passes -> dodge: dash across the sweep line INTO the new frame; you land exactly where your DPS is legal again.
  - 'CUTAWAY' (off-frame ambush): telegraph -> letterbox black bars slide in over ~0.8s, narrowing your visible screen -> execution: a ring of zombies + 2 slam waves (reuse SlamWave) spawn in the now-hidden top/bottom strips and march into frame -> dodge: the bars are a countdown; pre-position center-screen and dash the slam waves as they enter the visible band.
  - 'CLOSE-UP' (frame shrink): telegraph -> frame corner-ticks pulse red and the box shrinks toward the boss over ~1.2s -> execution: standing outside the shrunken box deals chip damage per second (you're 'out of shot') -> dodge: walk inward to stay framed — but the boss is now point-blank, so it's a deliberate risk-for-DPS squeeze.
  - 'JUMP CUT' (teleport, phase 3 only): telegraph -> screen does a 1-frame white flash + bars blink -> execution: boss instantly relocates the frame and itself to the opposite side -> dodge: no dodge, it's a repositioning demand — re-enter the new frame fast.
- **Exploits the combat model:** This is a pure shoot-only-while-still tax wearing a movie costume. Your gun has good range, but the FRAME caps your effective range AND gates the boss's vulnerability — so you can't just stand in a corner and plink. The frame constantly pans (forcing movement = forcing your DPS off) and the boss only takes damage while it shares the frame with your bullets. The optimal play is to plant yourself at the frame's leading edge, fire during the brief still windows, then dash to the next frame the instant it shoves. It punishes the corner-camper hardest, which is exactly the player who over-relies on shoot-while-still.
- **Phases:**
  - 100-66%: one big, slow frame. Gentle pans. Teaches 'damage only counts inside the box'.
  - 66-33%: frame shrinks ~30% and pans faster; CUTAWAY letterbox attacks begin. Now you're chasing the shot AND watching the bars.
  - 33-0%: 'FINAL CUT' — the projector flickers, the frame strobes between two positions (you must read which is solid via the corner-ticks), JUMP CUT teleports begin, and the film-grain off-frame fizzle radius tightens so only dead-center shots land. Screenshot moment: the whole arena is letterboxed and strobing like a broken film reel.
- **Counterplay:** A skilled thumb stops chasing the boss and instead chases the FRAME — treat the box as the real boss. Stand at its trailing edge so a pan brings the boss toward your line of fire, fire in the still beats, dash on the yellow 'ACTION' flash. Learns: 'my range and my damage are zone-gated; movement is repositioning into legal DPS, not retreat.'
- **Difficulty knobs + wave depth:** Frame size, pan speed, pan interval, off-frame fizzle radius, CUTAWAY ad count, CLOSE-UP chip DPS, FINAL-CUT strobe period. Debut wave 30+ (true finale). A wave-25 'matinee' variant: bigger frame, no JUMP CUT, slower pans.
- **Reward twist:** Drops a 'Director's Cut' relic: +25% gun range BUT your bullets gain a faint frame-tick trail — pure flavor callback. Also the death 'THE END' card doubles as the run-clear screen if it's the final boss.
- **Godot build notes:** Reuse SlamWave for CUTAWAY entries. Reuse the boss flash.gdshader. NEW: FrameBox.gd (Node2D with _draw drawing the rectangle + corner-ticks; a Tween pans/shrinks global_position and size; an Area2D child with a RectangleShape2D defines 'in-frame'). Gate boss damage by checking the frame Area2D overlaps the boss before applying take_damage (or set a `boss.in_frame` flag the boss reads). Cap bullet life outside the frame by having FrameBox keep a Rect2 the Bullet checks against (one global the bullet reads; if outside, queue_free). Letterbox bars = a CanvasLayer with two black ColorRects tweened in/out — keep it dumb. The 'CUT line' sweep = a thin reused SlamWave-style _draw band moving linearly instead of radially (small variant script).

### Chronos, the Stuttering Clock — a floating clock-face that drags YOUR time to a crawl while it moves at full speed, and rewinds the arena to where you just were
- **Spectacle:** A giant ticking clock-face hovers in the arena, hands spinning. When it attacks, the whole world visibly *stutters* — your bullets hang in the air mid-flight like beads on a string, your movement goes syrupy, the HUD timer flickers — while Chronos glides around untouched at full speed. In pixel art this reads as a hard, eerie 'everything-but-the-boss froze' moment; bullets frozen in a frozen arc is an unmistakable whoa.
- **Signature mechanic:** **Localized time dilation.** Chronos slows the PLAYER's time (movement, fire rate, bullet travel) without slowing itself or the trash. It is the only boss that attacks your *clock speed* rather than your position — making the fundamental tempo of your DPS the resource under attack.
- **Attack patterns:**
  - **Slow Field:** telegraph = a translucent clock-wedge (a pie slice of the clock face) sweeps across the arena like a radar hand over ~1.0s, clearly drawn -> execution = anything the wedge passes over enters 0.4x time for 2s (your bullets crawl, your strafe is molasses) -> dodge: dash OUT ahead of the sweeping wedge before it reaches you (dash burst is unaffected at its start, giving a clean escape) — or accept the slow and reposition early.
  - **Tick Rings:** telegraph = the clock 'ticks' — 12 faint dots light up around the dial one at a time, then all flash -> execution = thin damaging rings fire outward from the boss on each lit hour-marker as expanding arcs (reuse SlamWave ring) -> dodge: dash through the gaps between rings; trivial at normal speed, terrifying while you're inside a Slow Field, so the patterns are designed to layer.
  - **Rewind (phase 2+):** telegraph = the clock hands visibly spin BACKWARD and a faint ghost-trail of your last ~2s of positions draws as a dotted line -> execution = the player is yanked back along that trail to where they were 2s ago (no damage, pure displacement) -> counter: anticipate it — don't spend the pre-rewind window walking into danger you'll be rewound INTO; use the rewind by being somewhere safe 2s prior.
- **Exploits the combat model:** Shoot-only-while-still means your DPS is already gated on standing still; Slow Field then makes 'still' a trap — rooted to fire, your fire rate AND bullet travel are halved, so the same root that lets you shoot now makes your shots arrive late and thin. The genius is it doesn't stop you shooting, it makes shooting *slow*, so impatient players who refuse to move keep firing into the slow and accomplish nothing. Rewind specifically punishes kiting: the standard survivor strat is to walk a big circle, but Chronos snaps you back along that circle, collapsing the distance you bought.
- **Phases:** 100-66%: Slow Field + Tick Rings, slows are 0.5x. 66-33%: slows deepen to 0.35x and last longer; Rewind introduced. <33%: Chronos itself speeds up (its hands spin faster) and chains Slow Field directly into Tick Rings so you're dodging rings at crawl speed — the climax tests pure spatial reading.
- **Counterplay:** Skilled play = pre-position before each tick, dash out of wedges early (not late), and bank DPS in the windows BETWEEN slows rather than fighting through them. The lesson is tempo management: you stop treating 'stand and shoot' as the default and start treating fire windows as a resource you schedule.
- **Difficulty knobs + wave depth:** Knobs = slow factor, slow duration, wedge sweep speed, rewind lookback seconds, ring count. Debut **wave 20** (a tempo boss lands best once players have fire-rate/move relics to feel the loss).
- **Reward twist:** Drops **'Overclock' relic** — for the first 3s after standing still you fire at +time (a personal speed-up), the inverse of the boss's curse.
- **Godot build notes:** Reuse `Boss.gd` chassis and `SlamWave.gd` for Tick Rings. The Slow Field wedge = a `_draw()` rotating pie slice + an Area2D check. NEW small system: a **player-local time multiplier** — add a `time_scale: float` on Player/Gun/Bullet and multiply their own `delta` by it (do NOT touch `Engine.time_scale`, which would also slow Chronos and the trash). Slow Field/Rewind set `player.time_scale`. Rewind = a small ring-buffer of the player's `global_position` sampled a few times/sec; on rewind, Tween the player back to the buffered position over ~0.3s. Bullets already exist; just gate their movement on the shared multiplier.

### Vertigo, the Falling Room — a boss that grabs GRAVITY and tilts the whole arena, sliding you and everything else 'downhill' while it attacks the new low ground
- **Spectacle:** The boss raises an arm and the entire room *tilts* — the ground texture visibly slides, the camera cants a few degrees, and suddenly everything is sliding toward one screen edge as if the floor became a ramp. XP gems, trash enemies, and YOU all drift 'downhill' together. Periodically it flips the tilt to a new direction with a stomach-drop lurch. On a phone the whole world leaning and everything sliding the same way is a gut-level 'whoa, the floor is falling.'
- **Signature mechanic:** **Directional gravity drift across the entire arena.** A constant push vector is added to every actor each frame; the boss rotates that vector to attack the 'downhill' edge. It's the only boss that attacks by changing where 'down' is — you're no longer fighting on a flat plane, you're fighting on a tilting tray.
- **Attack patterns:**
  - **Tilt:** telegraph = a big translucent arrow (the new 'down' direction) sweeps in from a screen edge over ~1.0s and the ground starts visibly creeping -> execution = a steady drift pushes all actors toward that edge; the longer it holds, the faster the slide -> dodge: you can't dash it away (it's constant), you must walk UPHILL against it to hold position — but walking = not shooting. Or ride the slide and reposition.
  - **Downhill Slam:** telegraph = while tilted, SlamWave windup circles bloom along the LOW edge (where everything is sliding toward) -> execution = expanding rings detonate on the low edge exactly where the drift is carrying you -> dodge: dash perpendicular/uphill to break out of the slide before the rings land. The drift makes the dodge harder because it keeps pulling you back into the kill zone.
  - **Flip (phase 2+):** telegraph = the screen darkens for ~0.7s and a spinning arrow shows the NEW down direction -> execution = gravity snaps to a different edge, lurching everything the other way -> counter: re-orient immediately and re-walk uphill; mistimed players get carried into walls/hazards by the sudden reversal.
  - **Avalanche (phase 3):** telegraph = trash enemies all start sliding to one corner and pile up, glowing -> execution = the pile launches back UPHILL as a wave of bodies -> dodge: dash through a seam in the wave.
- **Exploits the combat model:** It directly weaponizes **shoot-only-while-still** by making 'still' nearly impossible to achieve for free. On a tilted floor, standing still requires actively walking uphill — which counts as movement — so you cannot hold position to fire without burning input on staying put, and the instant you stop to shoot, the drift slides you away (breaking your aim line and carrying you toward the Downhill Slam). To get a fire window you must find a spot where the drift is momentarily neutral (e.g. right after a Flip, mid-lurch) or accept being pushed. It also fights auto-aim-nearest indirectly: the drift bunches all trash to the low edge, so your auto-aim constantly snaps to the downhill pile instead of the boss.
- **Phases:** 100-66%: gentle, slow drift; long stable windows. 66-33%: faster drift, Flips begin, shorter windows. <33%: rapid Flip chaining + Avalanche, the floor is basically never flat — the test is finding micro-windows of neutral drift to land DPS.
- **Counterplay:** Skilled play = read the arrow, pre-walk uphill so you arrive at a neutral spot, and steal fire windows in the brief still moments right after a Flip when the drift hasn't ramped. Dash is reserved for escaping Downhill Slams, not for fighting the drift (you can't dash a constant force away). The lesson is that 'standing still' is a position you must EARN here, not a default.
- **Difficulty knobs + wave depth:** Knobs = drift strength, drift ramp rate, Flip frequency, camera tilt degrees (keep small for readability/comfort), Avalanche density. Debut **wave 30+** (a capstone that recontextualizes the movement-vs-DPS core). Keep visual tilt modest on mobile to avoid motion discomfort — sell it mostly through the sliding ground, not heavy camera roll.
- **Reward twist:** Drops **'Sure Footing' relic** — immunity to forced movement/knockback effects (you stop sliding), a permanent answer to the boss's signature.
- **Godot build notes:** Reuse `Boss.gd` and `SlamWave.gd` (spawn slam telegraphs along the computed low edge). NEW small system: a global **drift vector** the boss broadcasts; in each actor's `_physics_process` add `drift * delta` to velocity/position for Player, enemies, and XP gems (a 3-line addition each, or a shared autoload the Player/Enemy/XpGem read). Tilt visual = offset the existing `Ground` Parallax2D and apply a small `Camera2D` rotation Tween (keep <6 degrees). Flip = Tween the drift vector to a new angle. Avalanche = read enemies in the low corner, then set their velocity uphill for a burst. No new physics engine needed — it's additive velocity on systems that already move.


## Family: Arena Control (4)

### The Tide-Glutton — A bloated leviathan that floods the arena with a rising sea of acid, leaving only shrinking pixel-islands to stand and shoot from.
- **Spectacle:** A whale-sized sluglike mass that visibly INHALES the floor — the ground darkens cell-by-cell as glowing green acid wells up from the edges inward, lapping in animated ripples. By the end the whole screen is a luminous toxic sea with three or four little dirt islands poking out, the boss wallowing in the middle. On a phone the flood reads instantly: acid is a single high-saturation green wash with a bright animated waterline; the islands are warm tan against it. "The floor is literally gone" is the whoa.
- **Signature mechanic:** RISING FLOOD. The walkable arena permanently shrinks over the fight as acid encroaches from the borders. Standing on acid = continuous DoT (like ignite but unavoidable while you're in it). Only the islands and a narrowing central ring are safe. No other boss removes floor area like this.
- **Attack patterns:**
  - **Tide Pulse:** telegraph = the acid waterline flashes bright white and pulses outward 0.9s before it lurches -> execution = the flood line jumps inward one cell-row, eating the outermost safe ground -> dodge = you don't dodge it, you RELOCATE before it arrives; dashing across acid is the cheap escape but costs a tick of DoT.
  - **Geyser Spit:** telegraph = 3-4 faint filled circles (reuse SlamWave telegraph) bloom on the remaining safe islands -> execution = acid geysers erupt there, briefly flooding that island -> dodge = dash to a different island; this is the boss denying you the very footing it left you.
  - **Gulp Reset (phase move):** telegraph = boss rears up, screen-edge acid glows and recedes for ~1s -> execution = flood retreats halfway, re-opening ground, then immediately begins rising AGAIN, faster -> counter = the brief low-tide is your big DPS window, plant and unload.
- **Exploits the combat model:** Shoot-only-while-still demands a stationary footing, but the safe footings are exactly what's disappearing. As the sea rises you're herded onto smaller and smaller islands; on an island you CAN stand and pour fire (great), but Geyser Spit punishes camping by flooding your island, forcing a dash-relocation that silences your gun mid-burst. The boss turns "where can I even stand still" into the whole puzzle.
- **Phases:** 100-66%: flood rises slowly from edges, generous islands. 66-33%: Tide Pulses come faster and Geyser Spit activates, islands get hunted. 33-0%: "high tide" — only the central ring + one rotating island remain; Gulp Resets happen on a timer giving frantic low-tide DPS windows. Acid DoT scales up.
- **Counterplay:** Treat islands as firing platforms, the sea as transit. Pre-aim by positioning on the island NEAREST the boss so auto-aim locks on the instant you stop. Save dash for Geyser eruptions, not for crossing acid casually. Bank damage during every Gulp low-tide. The lesson: real estate management — defend your DPS platform, abandon it the instant it's telegraphed.
- **Difficulty knobs + wave depth:** Tune flood rise speed, acid DoT/sec, number & size of starting islands, Tide Pulse cadence, low-tide duration. Debut at WAVE 20+ (needs players who already understand the still-to-shoot rhythm). Easier variant: more islands, slower rise.
- **Reward twist:** Drops a relic "Tidewalker's Sole" — immunity to the FIRST second of standing on any hazard floor (gives a grace tick on all future arena-hazard bosses). Thematic and cross-boss useful.
- **Godot build notes:** Reuse SlamWave's faint-circle telegraph for Geyser Spit. NEW: FloodField — a coarse cell grid (e.g. 32px cells) as a PackedByteArray; one node's _draw() renders flooded cells as a green layer with an animated waterline; a canvas_item ripple shader sells motion cheaply. Player DoT = Area2D-free point-in-grid check each physics frame (reuse the ignite DoT path). Islands = small static Area2Ds tagged "safe_ground". Tween the flood radius/rows for Tide Pulse and Gulp Reset.

### The Cartographer — A floating ink-quill eye that redraws the arena live — erasing the floor into bottomless void and inking new platforms you must read like a map.
- **Spectacle:** A giant disembodied eye trailing an ink quill, hovering over a parchment-colored floor. It "draws": black ink blots spread across tiles and those tiles become VOID — a starfield/abyss showing through — while fresh ink-lines sketch NEW solid platforms elsewhere with a satisfying pixel-by-pixel reveal. Mid-fight the arena is a constantly redrawn archipelago of parchment over black void. On a phone it's stark and unmissable: solid = warm parchment, void = pure black with tiny stars. The whoa is watching the ground get un-drawn beneath you.
- **Signature mechanic:** VOID REWRITES. Sections of floor are erased into instant-death void and new floor is inked in, repeatedly remapping where you can exist. Unlike the Tide-Glutton's slow rise, this is fast, surgical, and READ-based — the eye telegraphs exactly which tiles it's about to erase by inking their outline first. Falling into void = big damage + knockback to nearest solid (not instakill, to stay phone-fair). No other boss deletes-and-creates floor.
- **Attack patterns:**
  - **Erase Stroke:** telegraph = a bold ink outline traces a stripe/blob of tiles ~1.0s before it fills -> execution = those tiles drop to void -> dodge = dash or walk onto solid before the ink completes; the outline IS the countdown (it fills like a loading bar).
  - **Ink a Bridge:** telegraph = dotted-line ink sketch shows a NEW platform forming over current void -> execution = it solidifies into standable floor -> counter = not an attack — it's an invitation; the boss often erases your current stand while drawing a fresh one, baiting a relocation.
  - **Blink-Scribble (phase):** telegraph = the eye SHUTS (screen briefly dims, ~0.7s) -> execution = on reopen, the floor layout has been completely redrawn to a new pattern -> dodge = during the dim, drift to arena center (statistically safest); on reveal, sprint to the nearest solid tile.
  - **Quill Lance:** telegraph = the eye narrows and a thin ink line points at the player (reuse SlamWave leading-band aimed shot) -> execution = a fast ink dart along that line -> dodge = dash sideways.
- **Exploits the combat model:** The arena is constantly being rewritten, so a "good still-spot" has a short shelf life — you get a stationary firing window only as long as your current tile stays inked solid. The eye weaponizes this by erasing the tile you're standing on the moment auto-aim has locked and you've committed to firing, forcing a panic dash that drops your DPS. Skilled play = predicting which tiles will survive and pre-standing there. Falling into void also yanks you to the NEAREST solid tile, which can reposition you closer to enemies and scramble your auto-aim target.
- **Phases:** 100-66%: small, slow Erase Strokes, lots of solid ground, learn to read ink outlines. 66-33%: Ink-a-Bridge + Erase combos that herd you across moving platforms; Quill Lance added. 33-0%: Blink-Scribble full redraws on a timer — the whole map reshuffles, demanding instant re-reading; solid area at its minimum.
- **Counterplay:** Read the ink. Bold outline = leaving now, dotted sketch = safe soon. Stand on tiles far from any active outline so you get clean fire windows. Treat Blink-Scribble dims as a free reposition to center. Dash is for committed escapes off an erasing tile, not for panic. Lesson: the floor is information — parse the telegraph, don't react late.
- **Difficulty knobs + wave depth:** Tune erase-stroke size/speed, fill (countdown) time, fraction of floor kept solid, Blink-Scribble frequency, void-fall damage. Debut at WAVE 25+ (highest read demand of the family). Easier variant: slower fills, never erases the player's exact tile without a bridge already drawn.
- **Reward twist:** Drops relic "Margin Note" — briefly shows a faint preview outline of the NEXT hazard on ALL future arena bosses (a half-second extra telegraph), rewarding the player who beat the hardest-to-read boss with better reads forever.
- **Godot build notes:** NEW VoidGrid: PackedByteArray over coarse cells, one _draw() paints void (black + cheap star noise) vs solid (ground texture). Each erase/ink is a Tween animating an alpha/fill mask so the ink-blot spreads smoothly. Player floor-check = point-in-grid each physics frame; on void -> apply damage + lerp to nearest solid cell center. Quill Lance reuses SlamWave's aimed leading band. Blink-Scribble = a full-screen ColorRect fade (CanvasLayer) covering the regrid.

### Conveyor Saint — A four-armed mill-god whose halo spins the entire floor into a giant conveyor belt, dragging you off your firing spot toward whirling blades.
- **Spectacle:** A serene golden idol with four arms and a slowly rotating halo, hovering above a floor that has become a visible CONVEYOR — chevron arrows etched across the ground all flow in one direction, then sweep around like a clock hand when the halo turns. Pixel art reads it perfectly: bright animated chevrons scrolling across the dark floor, plus spinning saw-disc sprites orbiting the edges. The whoa is watching the entire arena's arrows pivot at once.
- **Signature mechanic:** FLOOR DRIFT. A constant directional pull is added to the player's position every frame — the floor slides you. Standing still no longer means staying put; you drift, and drifting counts as the floor moving you, but YOUR feet aren't moving, so the gun keeps firing. The catch: the drift carries you toward the saw-discs ringing the arena. You must walk AGAINST the current to hold position — and walking = no shooting.
- **Attack patterns:**
  - **Belt Turn:** telegraph = the halo glows and rotates ~1.0s before the floor arrows begin swinging to a new heading -> execution = the drift direction rotates to where the arrows now point -> dodge = no dodge; you pre-walk to where the new current will deposit you, ideally a safe interior spot.
  - **Rim Saws:** telegraph = one edge of the arena flares red and saw-discs there spin up 0.8s -> execution = those discs surge inward a short distance -> dodge = dash perpendicular to the belt (dashing fights the drift for free).
  - **Halo Slam (phase):** telegraph = halo drops onto the boss, faint filled circle under it (reuse SlamWave) -> execution = expanding ring slam from center -> dodge = dash through the ring; tricky because the belt is shoving you while you time it.
- **Exploits the combat model:** This is the purest expression of "movement vs shooting." The floor drift means standing still still lets you shoot (feet idle) — BUT it slides you toward saws, so to survive you must actively walk against the current, which holds the trigger. The player is forced to choose every second: shoot and drift into danger, or walk to safety and deal zero DPS. Auto-aim-nearest also gets messy because the boss orbits and the drift constantly changes your range to it.
- **Phases:** 100-66%: single slow steady drift, learn to lean against it. 66-33%: Belt Turns rotate the current every few seconds; Rim Saws activate. 33-0%: drift speed ramps and the belt rotates continuously (a slow spin) so there's no stable still-spot — you're always managing position, snatching half-second fire windows when the current briefly points somewhere safe.
- **Counterplay:** Park in the "upstream" pocket where the current pushes you INTO open arena, not into saws — then you can stand and shoot while the belt holds you in place against a wall of nothing. Read the halo to pre-walk before each Belt Turn. Dash is free distance against the current — use it to reposition without spending a walk. Lesson: find the spot where the hazard works FOR your stillness.
- **Difficulty knobs + wave depth:** Tune drift speed, Belt Turn frequency & rotation amount, saw count/surge distance, whether the belt is stepped or continuously rotating. Debut at WAVE 15. Continuous-spin version is the hard mode.
- **Reward twist:** Drops relic "Counterweight" — a small permanent move-speed buff specifically while standing still-ish (anti-knockback flavor), making future drift/pull hazards easier to resist.
- **Godot build notes:** NEW DirectionalDrift: a Vector2 added to Player.global_position each _physics_process (and to loose XpGem positions so pickups slide realistically). The arrow field = one _draw() of scrolling chevrons, rotated by the current heading; Tween the heading on Belt Turn. Saws = Area2D + rotating sprite, reuse contact-damage. Halo Slam reuses SlamWave wholesale.

### Heart of the Maze — A pulsing crystalline core that grows living walls out of the floor, sealing you into ever-tighter corridors while its heartbeat tells you when to run.
- **Spectacle:** A massive glowing crystal heart at arena center, visibly throbbing — every beat sends a pulse of light outward and CRYSTAL WALLS erupt from the floor along the pulse lines, shattering up in jagged pixel shards. The open arena slowly becomes a glittering maze with the heart pulsing red at its core, you scurrying through narrowing glass corridors. On a phone it reads as bright crystalline barriers growing against the dark floor, and the heart's rhythmic glow doubles as a visible metronome. The whoa: the battlefield turning into a labyrinth in real time.
- **Signature mechanic:** GROWING WALLS that block line-of-sight AND bullets. Crystal walls rise from the floor, partitioning the arena into shrinking corridors. Crucially, walls block the player's auto-fire — bullets hit walls — and block auto-aim line-of-sight, so the boss can wall ITSELF off behind crystal, making your gun useless until you reposition for a clean lane. The walls also funnel the streaming trash enemies into chokepoints. No other boss in the family blocks projectiles/LOS with terrain.

  - **Heartbeat Grow:** telegraph = the heart flashes bright on its beat, faint outlines show where the next wall segments will rise ~0.9s ahead -> execution = walls erupt along those lines -> dodge = don't be standing on a spawning wall line (it shoves/damages you); step off the glowing outline.

- **Godot build notes:** NEW WallGrowth: wall segments as StaticBody2D (blocks player) + on the bullet collision layer (bullets hit & despawn) + occluding for the aim raycast — Gun's auto-aim should raycast LOS to the target and skip blocked enemies (small addition to existing nearest-enemy logic). Walls rise via Tween scale-y. Keep a grid of occupied cells with a flood-fill connectivity check so a path always exists. Crystal Lance & any aimed beam reuse SlamWave's leading band. Shatter shards = a particle/bullet burst on the existing enemy-projectile path.


## Family: Swarm / Summoner (4)

### Patient Zero — a parasite host that infects your trash mobs and turns the whole spawn stream against your DPS
- **Spectacle:** A hunched, ruptured corpse-thing trailing a sickly green spore mist, its back split open with a pulsing fungal bloom. It doesn't fight you much itself — instead, the normal zombies streaming in walk through its spore cloud and *change*: they bloat, sprout glowing green growths, and start moving differently, juddering and aggressive. Over a minute the entire screen of trash turns into a glowing-green infected horde. The 'whoa' is watching your own familiar trash mobs mutate into something worse in real time, the green spreading across the swarm like a visible plague — and realizing the boss made YOUR enemies stronger.
- **Signature mechanic:** **Contagion** — Patient Zero emits a slow-moving spore cloud (a roaming Area2D). Any normal enemy that touches it becomes *Infected*: +HP, +speed, it gains a death-burst spore puff that infects nearby enemies (the plague spreads on its own), and critically, infected enemies emit a fake 'priority' scent that biases auto-aim to pick THEM over the boss and over normal mobs. The infection is self-propagating: even if you kill Patient Zero, the plague keeps spreading through the survivors unless you clear infected bodies fast.
- **Attack patterns:**
  - *Spore Belch:* the host inflates and its bloom glows bright (telegraph ~1.0s, a swelling green tell) -> it coughs out a stationary spore cloud at a spot near the player that lingers ~5s, infecting anything that crosses it (including a sliver around you) -> dash out of the cloud's footprint; standing in it doesn't damage you directly but infects the mobs swarming you, so you want to lead enemies away from it.
  - *Burrow & Surface:* the host sinks into the ground leaving only a green mound telegraph (~1.2s) -> erupts under the player for a SlamWave-style ring hit, surfacing somewhere new -> dash off the mound. This is its only personal damage and its repositioning tool.
  - *Bloom Pulse (low HP):* the back-bloom pulses rhythmically, each pulse drawn as an expanding faint ring (telegraph per pulse ~0.7s) -> every pulse force-infects the nearest 2-3 enemies regardless of cloud -> there's no dodge; the counter is to have already thinned the horde so there's little left to infect.
- **Exploits the combat model:** The deepest auto-aim hijack in the set — infected mobs out-prioritize the boss for auto-aim, so the more the plague spreads, the more your gun *refuses* to shoot Patient Zero at all, locking onto an endless tide of buffed decoys. Shoot-while-still becomes a trap: infected mobs are faster, so rooting yourself to DPS lets the buffed horde collapse on you, but moving to survive shuts your gun off entirely while the infection only grows. It turns the game's core enemy stream into the boss's ammunition. The skill is choosing WHEN to ignore the boss and firebreak the infection vs. when to risk a rooted burst on the host.
- **Phases:** 100-66%: single spore cloud, slow spread, manageable. 66-33%: belches two clouds and infection death-bursts begin chaining (kill an infected = puff that infects its neighbors), so careless clearing accelerates the plague. 33-0%: Bloom Pulse active and the spread rate ramps — if you've let the horde grow, the screen is now a wall of green and the host is nearly unreachable by auto-aim; a controlled player will have kept the horde thin and can finally focus the host.
- **Counterplay:** The lesson is *firebreaking* — keep the overall enemy count low so the plague has no fuel, and accept that you sometimes kill the decoys to earn a clean line to the boss. Skilled play is rhythm: thin the horde, step so the boss is nearest, dump a rooted burst, reposition before the next cloud. Punishes hoarding enemies (which a greedy XP farmer wants to do).
- **Difficulty knobs + wave depth:** Debut at **wave 20** (needs the player to understand crowd control and have a clearing weapon). Tune: cloud lifetime/size, infection HP/speed buffs, spread tick rate, whether death-bursts chain, auto-aim priority weight of infected (the cruelest dial). Lowering the infected auto-aim priority makes it dramatically more forgiving.
- **Reward twist:** Its relic is a 'cure' themed item (e.g. converts a chunk of overkill damage into lifesteal). Bonus: if Patient Zero dies while zero infected enemies are alive on screen ('quarantine kill'), it drops a second relic.
- **Godot build notes:** New `InfectionBoss.gd` extending `Boss.gd`. Infected state = a flag on `Enemy.gd` that re-runs `configure()` with buffed stats, tints the sprite green, and sets a `decoy_priority` field the auto-aim target-picker adds as a weight (lowest score wins, infected get a bonus). Spore cloud = an Area2D that on `body_entered` for `enemies` group flips that enemy to infected. Death-burst = a one-shot small Area2D spawned in the enemy's `_exit`/death. Spread tick = a Timer on the boss. Burrow ring reuses SlamWave directly.

### The Chorus of One — a hooded conductor that copies YOU — every clone mimics your last few seconds of movement
- **Spectacle:** A faceless robed conductor floating above the arena, baton raised, surrounded by a slowly orbiting choir of empty hoods. When it sings (a silent scream telegraph), the hoods drop down and become **ghostly grey copies of the player's own sprite** that move *exactly as you just did, on a delay* — a literal echo of yourself swarming in. Picture four translucent Ryans all walking your own path a half-second behind you, fanning out as you turn. The 'whoa' moment: you dash to dodge, and a half-second later four copies of you dash through the same spot — your own movement becomes the swarm's attack pattern. Pixel art reads cleanly because the clones are recolored player sprites, instantly recognizable as 'me, but wrong.'
- **Signature mechanic:** **Echo Choir** — the boss records the player's movement into a ring buffer and spawns clone-adds that replay that buffer at staggered time offsets (e.g. -0.5s, -1.0s, -1.5s, -2.0s). The clones are enemies (they steal auto-aim, they deal touch damage) but they don't path to you — they retrace YOUR recent route. They also auto-fire weak shots back at you. Because they mimic you, predictable players spawn predictable, easily-herded clones; erratic players spawn chaos that traps themselves. You are, in effect, fighting your own past.
- **Attack patterns:**
  - *Crescendo (summon):* the conductor's baton sweeps and the orbiting hoods flash white in sequence (telegraph ~1.0s) -> the hoods drop and become echo-clones replaying your buffer -> no dodge; this is a setup move. The counter is to have spent the prior 2 seconds moving in a SAFE pattern (e.g. a tight loop) so the clones spawn into a clump instead of surrounding you.
  - *Downbeat:* the baton raises high and a metronome bar sweeps across the screen edge as a clean visual tell (telegraph ~0.8s) -> on the beat, every echo-clone AND the conductor fire a single bullet at your current position simultaneously -> dash on the downbeat to slip the volley (rhythm-game dash counter). Reading the metronome is the whole skill.
  - *Harmonize (low HP):* the conductor pulls all clones into a vertical line beside itself, all flashing in unison (telegraph ~1.2s) -> they sweep across the arena as a moving wall of bodies in formation, mimicking a single recorded dash of yours blown up huge -> dash through the one gap that corresponds to where you hesitated in the recording.
- **Exploits the combat model:** A mind-bending triple exploit. (1) Auto-aim-nearest: the clones spawn ON your own recent path, i.e. right next to where you are now, so they almost always become 'nearest' and steal every shot from the conductor — you literally cannot out-DPS yourself. (2) Shoot-while-still: standing still to fire makes your clones spawn standing still too, clustering on top of you as a damage stack, while the conductor sits safely behind its choir; you're punished for the exact rooting the gun needs. (3) Dash: every dodge you make is recorded and thrown back at you a beat later, so spamming dash floods the arena with dashing copies that cover the very escape routes you'll want next. The intended skill is *deliberate, legible movement* — you must move in clean shapes the clones can be herded into, the opposite of panic.
- **Phases:** 100-66%: 2 echo-clones at -0.5s/-1.0s offsets, Crescendo + Downbeat. 66-33%: 4 clones across -0.5s to -2.0s, Downbeat tempo speeds up (metronome sweeps faster). 33-0%: Harmonize unlocks, tempo doubles, and the conductor descends to player level briefly between songs — its only true vulnerability windows, when no fresh clones are out and your auto-aim can finally find it.
- **Counterplay:** A skilled thumb 'performs for the recording': moves in tight predictable loops to bunch clones, parks them in a corner so auto-aim and AoE clear them together, and saves dashes strictly for the Downbeat rather than spamming them. Teaches movement economy and that in a no-aim game your *path* is your real input — here it's literally turned into the enemy.
- **Difficulty knobs + wave depth:** Debut at **wave 25-30** (it's the skill-ceiling capstone of the family). Tune: number of clones, time offsets, buffer length, clone HP/touch damage, whether clones shoot back, Downbeat tempo, conductor descend frequency/duration. Disabling clone return-fire and using fewer/closer offsets is the big accessibility lever.
- **Reward twist:** Drops a unique 'Echo' relic that spawns ONE friendly afterimage of the player that fires alongside you for the rest of the run — you take the boss's gimmick and make it yours. Thematically perfect capstone reward.
- **Godot build notes:** NEW small system: a movement recorder on the player (or boss) pushing `global_position` into a fixed ring buffer every physics tick (~2s at 60fps = 120 entries, trivial). Echo-clones are a lightweight node that, each frame, reads `buffer[now - offset]` and lerps toward it — no pathfinding needed. They're in the `enemies` group so auto-aim/bullets treat them normally; recolor the existing player sprite grey/translucent via a `modulate` or the hit-flash shader inverted. Downbeat = a Timer-driven beat firing the existing enemy bullet at the player's current pos from every clone + conductor; metronome tell is a single `_draw()`/Tween bar. Harmonize and Crescendo telegraphs reuse the SlamWave flash-then-execute timing. The only genuinely new code is the ring-buffer recorder and the offset-replay read — both small.

### The Brood Mother — a bloated egg-sac queen who keeps the screen so crowded your auto-aim can never reach her
- **Spectacle:** A grotesque, pulsing sprite three tiles wide, her translucent abdomen visibly swelling and twitching with squirming silhouettes inside. Every few seconds she shudders and lays a clutch of glowing red egg pods that wobble, crack, and burst into skittering grubs. In pixel art she reads as a throbbing red mass that the screen-clutter literally flows out of — you SEE the swarm being born. The 'whoa': the moment you realize you've been dumping 100% of your DPS into a grub the whole time and her health bar hasn't moved.
- **Signature mechanic:** **Aggro Veil** — while ANY of her egg pods or grubs are alive within a radius of her, she is *untargetable* by auto-aim (a `boss_shielded` flag the target-picker respects). The swarm is literally her shield. Your gun, hunting nearest, will always lock the closest grub instead of her. To damage her you must physically clear a lane: walk/stand so the nearest enemy IS her, which means out-DPSing the spawn rate of the clutch.
- **Attack patterns:**
  - *Clutch Lay:* she rears up and her abdomen flashes white 3x (telegraph ~0.8s) -> drops 4-6 egg pods in a ring around herself -> pods are stationary for 1.2s then hatch into grubs. Counter: you can shoot pods before they hatch, but only if she isn't veiled by an earlier batch — so timing your clears matters; no dash needed, this is a DPS-priority puzzle.
  - *Skitter Surge:* all living grubs flash yellow in unison (telegraph ~0.6s) -> they all dash-lunge toward the player simultaneously -> dash perpendicular to scatter through the gap (dash counter). The lunge is the only thing that punishes standing still to shoot.
  - *Birth Spray (enrage only):* abdomen glows and inflates huge (telegraph ~1.0s) -> ruptures, launching 8 grubs outward in a radial burst like a SlamWave ring made of enemies -> dash through a thin sector to avoid the wall of bodies.
- **Exploits the combat model:** This is the purest auto-aim-nearest abuse in the set. The player's gun chooses its own target, and she weaponizes that by ensuring the nearest target is never her until you've manually butchered the swarm. Standing still to maximize DPS just feeds the auto-aim into expendable grubs while she heals/lays unmolested. The skill is *positioning your body* so the auto-aim's 'nearest' resolves to HER — you become the aim, with your feet, not your trigger.
- **Phases:** 100-66%: lays clutches of 4, slow cadence. 66-33%: clutch size 6, and grubs gain the Skitter Surge lunge. 33-0% (enrage): she stops moving, roots in place as a fortress, doubles lay cadence and adds Birth Spray — now it's a frantic race to clear faster than she breeds while dodging body-walls.
- **Counterplay:** A skilled thumb learns to ignore the boss bar early and treat the fight as crowd control: pick a corner, let auto-aim mow the grubs as they funnel, and only push for boss damage in the breathing window right after a clutch is cleared and before the next lay. Teaches lane discipline and that 'nearest target' is a resource you steer with your feet.
- **Difficulty knobs + wave depth:** Debut at **wave 10**. Tune: clutch size, hatch delay, veil radius (smaller = easier to get a clean shot past the swarm), Skitter Surge cadence, enrage lay-rate. Easiest knob for accessibility = shrink veil radius so a flanking step exposes her.
- **Reward twist:** Drops a relic plus, if you killed her in under a threshold time, a bonus 'unhatched egg' XP pickup worth a guaranteed level — rewards aggressive lane-clearing over turtling.
- **Godot build notes:** Egg pods and grubs are `Enemy.gd` variants spawned via the existing spawner with a `hatch` Timer. The Aggro Veil is a single new check in the auto-aim target-picker: skip any enemy in the `boss` group while it has `shielded == true`; flip that bool each frame based on a cheap `physics_overlap`/distance count of pods+grubs near her. Birth Spray reuses the SlamWave radial pattern but instances enemies along the ring instead of a damage band.

### Cleaver, the Halving — a chrome slime that splits into two perfect copies every time you nearly kill it
- **Spectacle:** A mirror-smooth liquid-metal blob with a single glaring eye. When you bring it low it doesn't die — it *flexes*, a seam of white light splits down its middle, and it tears itself into two smaller identical slimes that bounce apart with a wet metallic SHING. By the end of the fight the arena is a swarm of a dozen tiny scuttling chrome droplets all sharing one boss bar. The 'whoa': watching one boss become eight, the screen filling with skittering reflections of itself, the health bar a single shared sliver across all of them.
- **Signature mechanic:** **Mitosis Pool** — Cleaver and every fragment share ONE health pool (the boss bar = total HP across all bodies). Whenever any single fragment is reduced below a split threshold, instead of dying it spawns two half-size fragments with half its current speed-doubled and HP halved between them. Killing requires shrinking fragments past the minimum size where they can no longer split — only then do they actually pop. More fragments = more bodies stealing your auto-aim across the whole screen.
- **Attack patterns:**
  - *Split (passive/reactive):* a fragment's eye flashes white and the seam-line draws across it (telegraph ~0.5s) -> it splits into two that launch apart along the seam axis -> you can dash to stay clear of the launch, but the real counter is target discipline, not dodging.
  - *Quicksilver Lunge:* the smallest, fastest fragments periodically vibrate in place (telegraph ~0.6s, a jitter tell) -> then streak across the arena in a straight line toward the player like a bullet -> dash sideways out of the lane (dash counter). The more you've split it, the more lunges fire at once.
  - *Recombine (phase trick):* two nearby fragments pulse in sync and a tether line draws between them (telegraph ~0.9s) -> they slam together into a bigger, slower, tankier slime that recovers a split tier -> shoot the tether or dash between them to interrupt; if they merge, you lost progress.
- **Exploits the combat model:** Auto-aim-nearest scatter at its meanest — focus-firing 'the boss' is impossible because the boss is everywhere, and your gun keeps swapping to whichever droplet drifted closest, spreading your damage thin across the shared pool so nothing dies. Worse, shoot-while-still punishes you: to actually finish a fragment you want to stand and pour DPS, but the fast fragments lunge exactly when you're rooted, forcing a dash that resets your aim onto a different copy. The skill is herding fragments into a clump so 'nearest' keeps re-locking the same corner and you grind them below split-size together.
- **Phases:** It's HP-driven by nature — there are no scripted % gates, the phase IS the fragment count. Early: 1-2 big slow blobs, easy to track. Mid: 4-6 medium fragments, lunges begin, Recombine appears. Late: 8-12 minimum-size droplets that finally start actually dying — a frantic mop-up where every kill shrinks the shared bar visibly.
- **Counterplay:** A skilled player intentionally body-blocks fragments into a cluster against a screen edge so auto-aim stays locked on the pile, and times the kill-push for the gap between lunge volleys. Learning moment: against a shared pool, *concentration of fire* beats spreading it, and you control concentration with your position.
- **Difficulty knobs + wave depth:** Debut at **wave 15** (it's a target-discipline test, needs an established build). Tune: split threshold, minimum split size (lower = more total fragments = harder), lunge frequency, whether Recombine is enabled (off = much easier), fragment speed-up per split.
- **Reward twist:** If you kill all fragments within a short window of each other (a 'clean shatter'), they drop a shard relic with a bonus reroll; let it drag out and you just get the normal drop.
- **Godot build notes:** New `MitosisBoss.gd` extending `Boss.gd`. The split routine: on `take_damage` dropping a fragment below threshold, `queue_free` it and instance two scaled-down copies (half scale, half HP, +speed). Maintain a static/shared `total_hp` summed each frame to drive the existing boss bar via `health_fraction()`. Lunges and Recombine tethers use `_draw()` telegraph lines + Tween for the dash-streak, identical pattern to SlamWave's telegraph->execute timing.


## Family: Anti-Player Systems (4)

### STATIC, the Dead-Channel Broadcaster — A TV-headed ghoul that floods the screen with snow until your gun fires at GHOSTS instead of zombies.
- **Spectacle:** A hunched zombie whose head is a flickering boxy CRT television; the screen on its head shows a tiny laughing face made of scanlines. When it attacks, the WHOLE phone screen gets a CRT-snow vignette (rolling horizontal scanline bar, desaturation, edge static) and translucent blue 'ghost zombies' wink into existence around the player. In pixel art it reads instantly: the boss is the only thing with a glowing rectangle for a head, and real zombies are solid/red while ghosts are see-through/blue and jitter.
- **Signature mechanic:** DECOY HIJACK. STATIC spawns transparent 'broadcast ghost' enemies (1 HP, deal no contact damage) that sit closer to the player than the real horde. Because auto-aim targets the NEAREST enemy, your gun locks onto and empties into the nearest ghost — wasting your DPS on a phantom. Killing a ghost makes it 'change the channel' (pops with a static burst and respawns a new ghost elsewhere after 1.5s). No other boss steals auto-aim with fake targets.
- **Attack patterns:**
  1. Channel Flood -> telegraph: the CRT head flashes white and emits a visible expanding ring of scanlines -> execution: 3-5 blue ghost decoys fade in at mid-range around the player and the screen gets light snow -> dodge/counter: not a damage attack; counter is to MOVE so your gun stops firing, reposition so a REAL zombie becomes nearest, then stand still to dump damage into the boss/horde (dash to relocate fast).
  2. Signal Slam -> telegraph: a SlamWave filled-circle telegraph appears under the player tinted TV-blue with a 1.0s warning -> execution: expanding ring 'broadcast pulse' that hits for chunk damage AND spawns 2 ghosts on the band -> dodge: dash through/out of the ring (standard SlamWave counter).
  3. Dead Air (33% only) -> telegraph: head goes solid black, screen dims to near-black with only a center spotlight for 1.2s -> execution: heavy snow drops visibility to a small radius around the player and ghosts spawn aggressively -> dodge: no positional dodge needed, but you must hunt for real targets in the murk; dash toward the boss's glowing head (still visible) to keep DPS on it.
- **Exploits the combat model:** This is a pure auto-aim-nearest attack. The player never chose their target, so the boss chooses it FOR them by planting closer fake targets. The 'fight' becomes a spatial puzzle: the player must physically walk so that a thing-they-want-to-shoot is the nearest thing — which means lots of moving, which (via shoot-while-still) means lots of NOT shooting. It weaponizes the two core automations against each other.
- **Phases:** 100-66%: ghosts spawn 1 at a time, light/occasional snow. 66-33%: ghosts spawn in pairs and re-spawn faster; Signal Slam added. 33-0%: Dead Air engages, ghosts spawn in clusters of 3, and the CRT head briefly shows the player's own kill-count glitching — pure spectacle.
- **Counterplay:** Skilled play = treat ghosts as 'do not shoot' zones and reposition until a real enemy or the boss is your nearest target before planting your feet. Players learn to read transparency/color as 'priority' and to use dash purely for target-selection, not just dodging. Standing in a corner with the boss as the only nearby thing is the clean kill.
- **Difficulty knobs + wave depth:** Tune ghost count, ghost respawn delay, snow opacity (keep low enough to stay phone-readable), and how much closer ghosts spawn than real enemies. Debut around wave 15 (auto-aim manipulation needs the player to already trust auto-aim).
- **Reward twist:** Drops a 'Cracked Lens' relic: ghost/decoy enemies (its own and any future boss's) render at 40% opacity AND your gun deprioritizes targets below a HP threshold — a permanent partial counter to all decoy bosses.
- **Godot build notes:** Reuse SlamWave for Signal Slam. NEW: a 'BroadcastGhost' Area2D enemy in the 'enemies' group with 1 HP and a `is_decoy` flag + transparent jitter shader; and a small canvas_layer CRT-snow shader (scanline scroll + noise + vignette) toggled by the boss, alpha tied to phase. Ghosts use existing enemy spawn but with no contact-damage. Keep the shader cheap: a single noise texture scroll, no per-pixel raymarch.

### PUPPETINE, the String-Witch — A marionette zombie that SEIZES your joystick and walks you into her zombies like a doll.
- **Spectacle:** A tall, elegant rotting witch suspended from glowing crossed strings that vanish up off the top of the screen. When she 'hooks' you, four bright thin strings snap taut from her hands to YOUR character, and your survivor visibly does a little puppet-jerk. The strings are high-contrast white-on-dark and clearly anchor boss-to-player; on a phone you instantly see 'I am attached to her.' Cutting them flashes a satisfying snap + particle burst.
- **Signature mechanic:** JOYSTICK SEIZE (input override). Periodically PUPPETINE 'hooks' the player: for ~1.5s your joystick input is overridden — your character is dragged toward her (or her zombie cluster), and crucially while seized your gun also won't fire (you're 'moving' as far as shoot-while-still is concerned). It's not a reversal of controls (that's a different boss) — it's a full TAKEOVER: the boss moves you. The counter is a 'string-cut': during the hook, a DASH severs the strings and frees you. No other boss takes the joystick away entirely.
- **Attack patterns:**
  1. Hook & Reel -> telegraph: PUPPETINE raises both hands and four strings stretch out toward the player, glowing brighter over a 0.8s wind-up -> execution: strings snap taut, joystick is overridden, player is reeled toward a waiting zombie knot -> dodge: DASH during the hook = string-cut, snaps free (dash is the input that still works during seize). Fail and you get walked into contact damage.
  2. Tangle Field -> telegraph: faint string-cross lines (an X) draw on the floor in a zone with ~1.0s warning -> execution: stepping in the zone shortens your dash and adds a brief hook chance ('tangled') -> dodge: simply read and avoid the X-zones; positions the arena into safe lanes.
  3. Mass Pull (33%) -> telegraph: ALL strings flare and the screen edges pulse white, 1.2s -> execution: a long hook that reels you the full way to her unless cut, AND she yanks nearby zombies INTO a wall around you -> dodge: must dash-cut early; if you wait too long the dash only frees you partway. Teaches dash-timing precision.
- **Exploits the combat model:** Attacks the joystick itself — the one input the player thought was 100% theirs. Because being seized counts as movement, it also silences the gun, so a hooked player is dragged AND disarmed simultaneously. It elevates dash from 'dodge button' to 'the only input she can't steal,' making dash the literal lifeline. It punishes players who hoard dash for damage-dodges by demanding it for control-recovery instead.
- **Phases:** 100-66%: single hook every ~10s, long telegraph, easy cut. 66-33%: hooks come faster with shorter wind-up; Tangle Field zones appear. 33-0%: Mass Pull cycles and she sometimes double-hooks (a feint string then the real one) — the player must not waste the dash on the feint.
- **Counterplay:** Skilled thumb watches her HANDS, not the floor — the hand-raise is the true tell — and saves at least one dash charge specifically for cuts. Players learn dash economy and to read a feint. Pairs nastily with any boss but is a clean solo skill check.
- **Difficulty knobs + wave depth:** Tune hook frequency, wind-up time (keep >=0.8s for phone reaction), reel speed, dash-cut window, and feint chance in phase 3. Debut around wave 15-20. NOTE: only override movement input, never the dash input, or the boss becomes unfair.
- **Reward twist:** Drops 'Severed Strings' relic: your dash gains a small extra charge AND breaks any future hook/root automatically once per cooldown — directly themed to the lesson she taught.
- **Godot build notes:** Reuse SlamWave only for Tangle Field telegraph footprint. NEW (small): a `puppet_seized` state on the player controller that, while true, ignores joystick vector and lerps the player toward a target point, gates auto-fire off, and listens for the dash action to cancel (clearing the state + spawning a snap particle). Draw the strings with `_draw()` from boss hand positions to the player each frame. The whole override is ~30 lines gating the existing movement integrator.

### MIRRORMAW, the Wrong-Way Leviathan — A two-faced abyss-zombie that flips your world: drag left to walk RIGHT, and inverts the rule so you must MOVE to fire.
- **Spectacle:** A massive eel-like zombie with two faces — one normal, one a mirror-image grin — that periodically does a slow barrel-roll, and when it does, the ENTIRE arena visibly flips: the background mirrors, a glassy 'reflection' sweep wipes across the screen, and a big arrow-spiral icon flashes the current control state. In pixel art the flip is a hard mirror of the tilemap + a chromatic shimmer; the player FEELS the world turn over. Screenshot-worthy 'wait, which way is which?' moment.
- **Signature mechanic:** RULE INVERSION. MIRRORMAW has two distinct curses it toggles between (never both at unfair times): (A) AXIS FLIP — joystick directions are mirrored (left=right, and at higher phases up=down too); (B) FIRE INVERT — it flips shoot-while-still into shoot-while-MOVING: now your gun ONLY fires while you're moving and goes silent when you stand still. A persistent on-screen banner shows which rule is active. This is the marquee anti-player mechanic — directly editing the two laws the player has internalized. No other boss inverts a core rule.
- **Attack patterns:**
  1. Barrel Roll -> telegraph: the leviathan rears and a glassy reflection-wipe begins sweeping across the screen over ~1.0s, with the rule-banner pre-showing the INCOMING state grayed-out -> execution: at wipe completion the active rule toggles (Axis Flip on/off) -> dodge: not a hit; the 'dodge' is mental — re-orient your thumb. Movement during the wipe is normal, giving a beat to reposition.
  2. Undertow (during FIRE INVERT) -> telegraph: SlamWave rings spawn AT the player's feet repeatedly with normal warning -> execution: damaging rings that force you to keep dashing/moving -> dodge: dash out — and because FIRE INVERT is active, all that forced movement is actually KEEPING your gun firing. The boss turns its own pressure into your DPS window, a delicious inversion to discover.
  3. Hall of Mirrors (33%) -> telegraph: the screen splits with a vertical mirror line, 1.2s, BOTH axis flip and a horizontal flip arm -> execution: full inversion (left=right AND up=down) plus FIRE INVERT simultaneously for a short burst -> dodge: survive the disorientation by moving in slow deliberate arcs; standing still is now both safe-from-overshoot and a DPS-loss.
- **Exploits the combat model:** The boldest possible attack on the control scheme: it rewrites shoot-while-still and the joystick mapping themselves — the two things the entire game trained the player to trust. Under FIRE INVERT, the optimal behavior literally reverses: the cautious 'plant and shoot' player deals ZERO damage and must learn to fire-by-kiting. Under AXIS FLIP, every dodge instinct betrays them. It's the ultimate graduation exam for the combat model.
- **Phases:** 100-66%: only AXIS FLIP (horizontal), banner always shows state, long telegraphs — a gentle introduction. 66-33%: introduces FIRE INVERT phases (and Undertow), toggling between the two curses with clear wipes. 33-0%: Hall of Mirrors stacks both, flips come faster, and the banner blinks instead of staying solid — final-exam chaos but still readable.
- **Counterplay:** Skilled thumb reads the BANNER every flip and consciously re-maps: under fire-invert, adopt a 'figure-8 kite' to keep firing; under axis-flip, move in small committed taps rather than panic-swipes. The player learns that the game's rules are just defaults — true mastery is adapting. Hardest, most memorable boss.
- **Difficulty knobs + wave depth:** Tune flip frequency, telegraph length (NEVER below 1.0s — disorientation needs reaction room), whether vertical flip is included, and banner persistence. This is a CAPSTONE boss: debut at wave 25-30+. Keep it rare so the flip stays special.
- **Reward twist:** Drops 'The Other Eye' relic: lets the player TAP the banner to manually pick which of two minor buffs (movement-fire window vs still-fire window) is active for 5s — a tiny taste of controlling the rule themselves, themed to beating the rule-changer.
- **Godot build notes:** Reuse SlamWave for Undertow. NEW (small, just flags): an `axis_invert: Vector2(-1,1 / -1,-1)` multiplier applied to the joystick vector in the movement integrator, and a `fire_while_moving` bool that inverts the existing shoot-while-still gate. The barrel-roll = a Tween that flips the camera/tilemap node's scale.x (and .y in phase 3) + a CanvasLayer reflection-wipe shader. A persistent Control 'rule banner' panel reads the two flags. The mirror visual is the only art cost; the mechanic itself is two booleans.

### OVERCLOX, the Heat Tyrant — A molten reactor-zombie that turns YOUR own auto-fire into a bomb: keep shooting and your gun overheats and jams.
- **Spectacle:** A bloated zombie fused to a cracked reactor core; veins of glowing orange lava pulse brighter the more YOU shoot. The arena floor tints redder as a shared 'heat' rises, and when your gun overheats it visibly belches a puff of black smoke, the muzzle goes dark, and a red HEAT bar above the player flashes JAMMED. In pixel art the boss is a black silhouette with animated lava cracks; unmissable.
- **Signature mechanic:** WEAPONIZED AUTO-FIRE / HEAT JAM. While OVERCLOX is alive, every shot your auto-fire makes adds to a Heat meter (drawn as a thin bar under the player). Sustained standing-still fire fills it fast; it cools only while you are NOT firing (i.e. while moving). Max heat = your gun JAMS for ~2s (no fire even when still) and vents a small self-damage burst. The boss has so much HP you can't out-burst it — it forces you to fire in disciplined bursts. No other boss touches the auto-fire trigger / forces a self-imposed fire ration.
- **Attack patterns:**
  1. Meltdown Pulse -> telegraph: the boss's lava cracks flare and a SlamWave circle blooms under it; simultaneously the global Heat bar gets a yellow 'danger' overlay -> execution: a damaging heat ring + your Heat meter gets a flat +chunk -> dodge: dash out of the ring (and the dash, being movement, also cools your gun — a nice combo to teach).
  2. Solar Flare -> telegraph: four thin red lines sweep out from the boss like clock hands, rotating, with a 1.0s pre-spin tell -> execution: the rotating beams deal contact damage AND any beam crossing the player spikes Heat -> dodge: stand in the gaps between sweeping lines, dash across a beam if cornered.
  3. Forced Vent (33%) -> telegraph: boss turns white-hot, screen-edge gets a red pulse, 1.2s -> execution: it instantly maxes your Heat meter, jamming your gun for the duration, then hurls a slow wide ring -> dodge: pure movement phase — you literally can't shoot, so you dash/kite the ring and reposition, then resume once the jam clears.

The game's default optimal play is 'plant your feet and let the gun work' — OVERCLOX makes that the LOSING line, because uninterrupted standing fire overheats and self-jams. It forces the player to voluntarily MOVE (turning off their own DPS) to cool down, converting shoot-while-still from a free advantage into a resource you must spend carefully. It also makes dash dual-purpose: dodge AND coolant.
- **Phases:** 100-66%: heat builds slowly, generous cool rate, Meltdown Pulse only. 66-33%: heat builds faster, Solar Flare added, cool rate drops. 33-0%: Forced Vent cycles every ~12s and ambient heat rises on its own even while you move — the fight becomes a frantic burst-and-scoot rhythm.

### CHRYSALIS, THE NINE-WAVE MOTH — a cocoon that you must NOT fully kill on time — interrupt its hatch and it emerges weak, let it ripen and a bullet-hell moth-god is born

The genius is the moth phase inverts it: a sustained bullet-hell where staying still = death, so your auto-fire is throttled to whatever scraps of stillness you can find between bullet waves. The hatch race means a greedy player who keeps moving safely during the cocoon gets PUNISHED with a stronger moth — the game literally rewards standing in danger.

Mastering the cocoon trivializes the moth.
- **Difficulty knobs + wave depth:** Debut wave 15 (needs a player with a built weapon to win the race). Knobs: hatch-bar duration, cocoon HP, the HP->tier thresholds, moth bullet speed/density per tier. Lower bullet speed for early debut; tighten the spiral for wave 25+ remixes.

### The Tide (Chase-the-Wall Set-Piece) — A roaring wall of death scrolls across the arena and never stops — you must flee ahead of it forever while it forces you to choose between running and shooting.
- **Spectacle:** One edge of the screen becomes a churning, glowing wall — a tsunami of black goo / a rolling firestorm / a collapsing void (pick the skin) — that slowly grinds forward across the whole field, the camera panning with it so the ground scrolls endlessly. Sparks and debris fly off its leading edge. Behind it, the world is gone (solid black). On a phone it reads as an unmistakable bright screen-spanning bar marching toward you with a hard contrast edge. The boss is the wall.
- **Signature mechanic:** **A persistent, screen-spanning instant-death wall that scrolls at a fixed speed, forcing a continuous chase.** Unlike the worm's shadow (which homes), the Tide is a flat moving line you cannot dodge sideways past — you can only stay AHEAD of it. The arena visually scrolls (forced camera pan) so there's always more ground ahead but the wall always advances. The boss's vulnerable "heart" rides at a fixed offset inside the wall face; killing it stops the tide. No other boss uses a non-homing scrolling kill-plane.

  - **Surge:** the wall edge flares brighter and a chunk of it bulges forward (telegraph: a section of the edge glows + a drawn bulge outline, ~1.0s) -> that section lunges ahead faster for a moment -> dash forward or laterally out of the bulging lane.
  - **Debris Spit:** the wall hurls 2-3 chunks ahead of itself as arcing projectiles with landing-circle telegraphs (SlamWave tells) ~0.9s -> they slam down in your escape path -> dash around the marked spots WITHOUT slowing your overall retreat (a debris hit doesn't kill but staggers/slows you into the wall — the real threat).

The Tide tunes its speed so that you can only afford SHORT stops — every burst at the heart costs ground. The player is constantly solving "how many bullets can I afford before the wall eats me?" Auto-aim helps (the heart is usually nearest since you're fleeing INTO open space ahead), but trash adds spawned BETWEEN you and the wall steal your aim backward — punishing you for turning around at all. Dash is your get-out-of-jail when a stop runs long.
- **Phases:** 100-66%: steady advance, occasional Debris Spit. 66%: wall speeds up ~20% and Surges begin — the safe stop-window shrinks. 33%: the wall splits into TWO offset bands with a weaving safe lane between them (you must serpentine forward through a gap that drifts left-right), and the heart only exposes during the gap-aligned moments — peak "run the gauntlet" spectacle.
- **Counterplay:** Build a rhythm of micro-stops: sprint to open a buffer, plant for a half-second burst on the heart, sprint again. Skilled players read the Surge tell and bank dashes for it. Relics that boost move speed or fire rate directly extend your safe-stop budget — a satisfying build-payoff fight. Teaches buffer management and the value of move-speed.
- **Difficulty knobs + wave depth:** Wall advance speed, surge frequency/magnitude, heart HP (= total stop-time budget), debris count, the gap width in phase 3, how far ahead open ground extends. Debut at **wave 20** (needs a player who already respects the move/shoot tradeoff).

### Mirrorman — a rushdown clone that copies YOUR movement on a delay and spawns decoys that hijack your auto-aim
- **Spectacle:** A glassy chrome silhouette of the player character, flipped and palette-inverted, trailing a smear of shattered-mirror shards. It moves EXACTLY like you did one second ago — same joystick paths, same dashes — so the player has the uncanny "that thing is ME" moment. When it splits, the arena fills with translucent reflections of your own sprite, a hall-of-mirrors that's instantly readable: they all look like you, ghosted and semi-transparent.
- **Signature mechanic:** **Delayed movement-mimic + auto-aim decoys.** Mirrorman records your movement and replays it ~1s behind, so it mirrors your pathing and effectively closes whatever distance you opened (you can't simply out-kite a copy of yourself). On top of that it periodically spits out 2-3 transparent **decoy clones** of you that don't deal damage but ARE in the enemies group — so your auto-aim-nearest gun locks onto the nearest decoy and wastes your entire still-window shooting a harmless mirror while the real Mirrorman closes in. No other boss attacks the auto-aim targeting itself.
- **Attack patterns:**
  - Echo Step (passive): no telegraph — it walks your 1-second-old path. The counter is to make your OWN recent path a bad place for it to be; jagged movement feeds it jagged movement.
  - Shatter Spawn: telegraph -> Mirrorman flashes and a ring of mirror-cracks appears around it (~0.7s) -> 2-3 decoy-yous burst outward and scatter toward you -> dash THROUGH the decoys to reposition so the real boss (the solid, opaque one) becomes your gun's nearest target again.
  - Reflected Lunge: telegraph -> a bright inverted-color line-arrow snaps from Mirrorman to where YOU were a beat ago (it aims at your echo, not you) -> it dash-strikes that stale point -> counter by being somewhere NEW: if you keep moving fresh, its lunge lands on your ghost.
- **Exploits the combat model:** It's a two-pronged assault on the auto-systems. (1) auto-aim-nearest: decoys are nearer harmless bait, so the gun throws your DPS at fake targets — the player must physically reposition so the REAL boss is nearest before a still-shot counts. (2) the movement-mimic specifically counters kiting: against most rushdown you open distance then plant and fire, but a delayed copy of your own movement keeps pace, so the safe "create space then shoot" loop fails. You must break your own pattern — move unpredictably, then snap still in a spot the mimic can't be. Dash is used to punch through decoy walls to restore targeting.
- **Phases:** 66% — decoy count rises to 4-5 and they linger longer, choking auto-aim harder. 33% — the mimic delay SHRINKS from 1s toward ~0.4s, so the copy is right on top of you and reacts almost in real time; decoys now briefly mirror-fire a single harmless tracer to add visual noise. Visual chaos peaks: a dozen translucent yous on screen — the "what IS this" screenshot moment.
- **Counterplay:** Learn to tell the solid opaque real boss from translucent decoys at a glance (high alpha contrast). Skilled play: move in fresh, non-repeating bursts to desync the mimic and feed its lunges your stale ghost; dash through decoy clusters to clear targeting; fire only when the real boss is unambiguously the nearest body. Teaches target discipline and anti-pattern movement.
- **Difficulty knobs + wave depth:** Tune mimic delay (smaller = harder), decoy count, decoy lifetime, and decoy spawn cadence. Decoy opacity is a READABILITY knob — keep decoys clearly more transparent than the real boss. Debut at wave 20 (auto-aim manipulation is an advanced mind-bender). 
- **Reward twist:** Drops a **Mirror Shard relic** — spawns ONE friendly decoy of the player that taunts/pulls nearby trash auto-aim toward itself (turns the boss's own trick into a player tool).
- **Godot build notes:** NEW: a small recorder — append (position, velocity) to a fixed ring buffer each physics frame; the boss reads the entry `delay` frames back and moves there (replaces Boss.gd chase). Decoys are cheap Area2D + player Sprite2D (low alpha, no damage) added to group "enemies" so the existing nearest-target auto-aim picks them up for free — no aim code changes needed. Reuse SlamWave.gd leading-band for Reflected Lunge, aimed at a stale buffer entry instead of live player. Use the hit-flash/inverted palette via a simple canvas_item shader (negate RGB) to distinguish boss vs decoys.

### Hauler & the Hooks — a hit-and-run harpooner that NEVER stays in range — it tags you with a chain and reels you off your firing spot
- **Spectacle:** A hunched, hulking shape dragging clanking chains, hooks dangling from every limb, that strafes the FAR edge of the arena and never comes close. When it fires, a taut bright chain snaps across the whole screen from the boss to you — a single thick high-contrast line you can't miss on a phone — and then you are visibly YANKED across the floor, your character skidding with a dust trail. The screenshot moment is the chain stretched corner-to-corner with you sliding helplessly along it.
- **Signature mechanic:** **The Reel** — Hauler stays out of auto-aim range and lands a hook that TETHERS the player and drags them (a brief movement-override pulling you toward the boss or a hazard). While reeled, your joystick is fighting the pull and you are by definition MOVING — so your gun is silenced exactly when you'd most want to retaliate. It's the only boss that directly seizes the player's movement to deny fire, rather than just out-running them.
- **Attack patterns:**
  - Harpoon Cast: telegraph -> Hauler rears back and a thin dotted aim-line extends from it to your current position over ~0.9s (a clear dashed laser-sight on the floor) -> a hook fires along it; if it connects, the chain goes taut and you're reeled ~40% of the way to the boss over ~0.8s -> dash sideways off the dotted line during the wind-up to break the cast (a clean dodge), OR dash the instant it connects to snap the chain early.
  - Drag-to-Hazard: at higher HP it casts toward you but reels you toward a SlamWave-style ground ring it's already telegraphing elsewhere — telegraph is the dotted hook-line PLUS a faint danger circle at the destination -> you're hauled into the blast unless you dash to sever -> double-read: dodge the reel direction, not just the ring.
  - Hit-and-Run Strafe (passive): no telegraph — Hauler constantly circles at max range so auto-aim barely reaches it; it ducks out the moment you start landing hits. The "attack" is denial of range.
  - Anchor Slam (close-range punish, late): if you DO corner it, it spins the chains in a short bright expanding ring (SlamWave) to shove you back out to harpoon range -> dash through the gap.
- **Exploits the combat model:** Triple threat to the fire economy. (1) Range: by hugging the arena edge it sits at the limit of auto-aim-nearest, so trash adds (which spawn closer) constantly steal your lock — you must clear adds or walk INTO danger to make the boss your nearest target. (2) The Reel directly converts your shoot-while-still into forced movement: a successful hook guarantees ~0.8s of zero DPS while you skid. (3) It punishes commitment: the moment you plant to shoot is the moment you're most hookable, so standing still to fire is itself the bait. Dash is your hook-break AND your re-approach tool, constantly contested.
- **Phases:** 66% — fires TWO hooks in quick succession (dash the first, the second catches greedy re-plants); strafe speed up. 33% — Drag-to-Hazard becomes the default cast and it lays a hazard ring before every reel, so a missed dash means hook + blast. Final stretch: a panic "Reel-In Frenzy" — rapid short hooks that keep you perpetually skidding, a desperate rushdown where you must dash on a rhythm to ever touch the trigger.
- **Counterplay:** Respect the dotted line — dash perpendicular to it during the ~0.9s cast, every time. Skilled play: manage adds so the boss stays your nearest target, bait casts then punish the post-cast recovery with a planted burst, and save dash specifically for hooks (not chip). The player learns range control and that dodging the PULL VECTOR matters as much as dodging a blast.
- **Difficulty knobs + wave depth:** Tune cast wind-up (your reaction window), reel distance/duration, strafe range vs auto-aim range, hook cadence, and add-spawn pressure. Reel duration is the core fire-denial knob. Debut at wave 25 (range + tether literacy is advanced). A two-hook variant at 30+.
- **Reward twist:** Drops a **Grapple relic** — the player's dash now leaves a brief tether that yanks the player TOWARD the nearest enemy on the next dash (turn the hook against them, great for closing on future hit-and-run bosses).
- **Godot build notes:** Chain = Line2D between boss and player, made taut on hit. Cast telegraph = dashed _draw() line from boss to player (reuse SlamWave's telegraph-then-commit timing). The Reel is a NEW small movement-override: a bool `reeled` + anchor point; while true, in player `_physics_process` blend velocity toward the anchor and ignore most joystick input — but a dash sets `reeled=false` and snaps the Line2D (the universal counter is preserved). Strafe = Boss.gd chase with a min-distance "keep at range" rule (move away if player too close). Anchor Slam and Drag-to-Hazard reuse SlamWave directly.

### Vel, the Quickening — a duelist that gets permanently faster every second you survive — the longer the fight drags, the less you can ever stand still
- **Spectacle:** A lean, knife-thin pixel figure wrapped in a smear of motion-blur after-images. Early on it walks; by the end it is a screaming streak of 3-4 cyan ghost-silhouettes dragging behind it, leaving a fading trail across the arena. The screen edges slowly bleed a cyan vignette that intensifies with its speed, so the WHOLE PHONE visibly "speeds up" — a clear, escalating "oh no, it's getting faster" read even on a small screen.
- **Signature mechanic:** **Quickening** — Vel's chase speed ramps continuously and PERMANENTLY the entire fight (e.g. +X%/sec, no cap until phase gates). Nothing resets it. This is the ONLY boss whose threat is a monotonic clock: every second you don't kill it, it gets closer to out-running your ability to ever park and shoot. The trail-ghost count and vignette are a direct readout of current speed.
- **Attack patterns:**
  - Sidewinder Dash: telegraph -> Vel freezes for ~0.6s and draws a bright cyan line-arrow from itself through your current position (SlamWave's leading-band, reshaped into a thin lance) -> it blitzes along that line at 3x its current speed -> dodge by dashing perpendicular to the arrow; the line is the hint of WHERE, the freeze is your shoot window.
  - Cross-Cut: telegraph -> at higher speed it stamps TWO crossing line-arrows (an X centered slightly ahead of you) over ~0.8s -> dashes one then immediately the second -> dash to a quadrant gap, not backward.
  - Pace Press (passive): no telegraph, it simply closes distance. The "attack" is denial — once it's inside auto-aim range and moving, your gun never gets a still moment.

Skilled play: pre-stack your highest-burst weapon, kite in wide circles to keep distance early while landing free still-shots, then bait the freeze-dashes — every telegraph is a guaranteed ~0.6s you can plant and unload. Dash is spent ONLY on the line/X telegraphs, never wasted on chip. The lesson: against rushdown, the telegraph freeze IS your fire window — provoke it.

- **Godot build notes:** Reuse Boss.gd chase (just feed it a growing speed var). Ghost-trail = cheap GPUParticles2D or a ring buffer of fading Sprite2D copies. Vignette = a fullscreen CanvasLayer ColorRect with a simple radial canvas_item shader, alpha driven by current_speed.

### The Metronome — a rushdown duelist locked to a visible musical beat — it ONLY moves and strikes on the downbeat, so you fight inside the rhythm
- **Spectacle:** A tall conductor-marionette of brass and glass with a swinging pendulum for a torso. The whole arena pulses in time — a row of beat-pips marches across the bottom of the screen and the boss flashes white on every downbeat. It teleport-stutters position in crisp quantized hops (no smooth glide), so it reads as a stop-motion dancer. When it speeds the tempo, the music and the screen-pulse visibly accelerate: an unmistakable "the beat is getting faster" panic, perfectly legible on a phone via the pip bar.
- **Signature mechanic:** **Beat-locked motion** — the Metronome is FROZEN between beats and only acts ON the beat (move-hop, dash-strike, or summon, one action per beat). Time is quantized. Crucially this means there is a guaranteed, rhythmic STILL window between every beat — but it's exactly long enough to bait you into shooting and then get clipped if you misread which beat is the strike. No other boss gates its entire behavior to a shared clock the player can hear and see.
- **Attack patterns:**
  - Tick-Hop: on off-beats it teleport-hops one step toward you (faint pre-flash on its destination tile ~0.4s ahead) -> appears there -> no dodge needed, this is repositioning, but it tells you the rhythm.
  - Downbeat Lunge: telegraph -> on the bar's final pip it raises the pendulum and a bright bar-arrow snaps toward you (SlamWave leading-band, beat-synced so the warning length == one beat) -> on the downbeat it dash-strikes the full line -> dash on the downbeat, not before; mistime by half a beat and you eat it.

- **Exploits the combat model:** This is a TIMING puzzle layered on shoot-while-still. It turns the player's fire economy into a metronome game: greedy players who stand still on a strike beat die; players who never commit to a beat never shoot. Dash is rhythm-gated — it must land on the downbeat to clear the lunge, punishing panic-dashing. The whole fight teaches you to fire in the gaps of a clock.
- **Phases:** 66% — tempo increases ~30% (pips march faster, still-windows shorten). 33% — adds an OFF-beat fakeout: occasionally it flash-telegraphs a strike but acts a half-beat late (a syncopation feint) — punishes pure muscle-memory and forces you to actually read the pendulum, not just the pip. - **Reward twist:** Drops the **Downbeat relic** — for the rest of the run, your gun fires a free bonus volley every few seconds on a subtle internal beat, rewarding players who time their standing-still to it.
- **Godot build notes:** NEW: a tiny BeatClock autoload — a repeating Timer emits `beat(index)`; the boss subscribes and gates every action to it; the pip HUD is a Control with N TextureRects lit by beat index. Reuse SlamWave.gd for the lunge (telegraph duration = beat period). Hit-flash shader fires on every downbeat.


## Family: Economy / Greed (4)

### The Croupier — a giant floating roulette-wheel dealer that turns the whole arena into its casino table and bets your build against the house
- **Spectacle:** A tuxedoed skeletal dealer the size of a building, lower half a spinning red-and-black roulette wheel that IS the arena floor — the ground under the player is divided into glowing pie-slice sectors that slowly rotate. A giant ball clatters around the rim. Pixel art reads beautifully: bold red/black wedges, a bouncing white ball, green felt, and a grinning chrome dealer. When it 'pays out' the whole screen rains golden chips. A genuine 'what IS this' moment — you're standing on a live roulette wheel.
- **Signature mechanic:** THE WAGER. A few times per fight the Croupier slams a chip down and forces a bet: a small non-blocking banner offers a stake — e.g. 'DOUBLE OR NOTHING: fire rate' — and a short window where you pick by standing in a red, black, or green floor sector (movement = your vote). The wheel then spins and resolves: win and the staked stat is buffed for the rest of the fight; lose and it's halved until the boss dies. Green (rare) is all-or-nothing: win = drop a bonus relic on its death, lose = your auto-fire is disabled for 6 seconds. You are literally gambling your build mid-fight.
- **Attack patterns:**
  1. House Edge (sector hits): two of the rotating floor sectors flash red for ~1.0s -> those wedges erupt in a pillar of chips/fire -> dash to a black or unlit wedge before they fire. Because the floor rotates, the safe wedge is a moving target you must track.
  2. Ball Drop (telegraph -> execution): the rim ball glows and speeds up, a dotted predicted-landing marker arcs to a floor tile near you ~1.1s out -> it slams down as a SlamWave ring from that point -> dash out of the ring; the marker telling you exactly where lets a calm thumb pre-position.
  3. Chip Rake (auto-aim abuse): the dealer sweeps an arm and scatters a line of glittering high-value chips across the floor between you and itself, each tagged as an enemy. Auto-aim and your bullets get raked onto the chip line instead of the boss; the chips are worth real coins if cleared, but every one you leave standing shields the Croupier behind them.
  4. (high HP only) Jackpot Spin: the entire wheel speeds up and ALL sectors strobe in a rotating safe-band pattern (one clean wedge sweeps around like a clock hand) -> stand on the moving safe wedge and ride it; everything else fires. Pure choreography.
- **Exploits the combat model:** The Wager weaponizes the player's stats directly — a 'lose' on fire rate or damage gutts the very auto-fire the whole game is built on, and the green-loss outright DISABLES auto-fire for 6s, the harshest possible debuff in a no-manual-fire game. Sector hits and the rotating safe wedge force constant repositioning, which by the shoot-while-still rule means you can't fire while you're surviving — you bank DPS only in the still beats between spins. And Chip Rake exploits auto-aim-nearest to literally bet your bullets: greedy players clear the chips for coins but expose themselves; safe players ignore them but the boss hides behind its own winnings.
- **Phases:** 66% HP: single Wager offered, House Edge + Ball Drop only. 33% HP: Wagers come faster and stake bigger (double-or-nothing on two stats at once), Chip Rake added, green bets appear. Below 15%: Jackpot Spin — the floor never stops rotating, and it offers one final 'ALL IN' wager (stake everything for a guaranteed double-relic drop, or lose all temp buffs). The arena becomes a spinning bullet-hell casino.
- **Counterplay:** Learn that the wheel's rotation is readable and constant-speed, so safe wedges are predictable. Treat Wagers as deck-building on the fly — a confident player takes double-or-nothing when their build can carry a loss; a fragile player declines by standing in a neutral wedge. The lesson: in a greed boss you manage variance, and your own stat sheet is now ammunition the boss can borrow.
- **Difficulty knobs + wave depth:** Tune wheel rotation speed, sector-hit count, Wager frequency/stake size, green-bet odds, and the auto-fire-disable duration on green loss. Debut at wave 20 (it presupposes the player understands their build well enough to bet it). Encore at wave 40 with the floor always spinning.
- **Reward twist:** Death payout is itself a final spin: the boss drops a relic AND triggers one free no-downside bonus spin (a slot-style reel of 3 relic icons) — match-3 for a rare relic, otherwise a common. Plus any 'won' wagers persist as a one-run buff. Losing wagers cost nothing after death.
- **Godot build notes:** Arena wheel = a full-screen _draw() overlay of rotating colored sectors (angle from a single rotating float), with Area2D zones per wedge for hit/safe detection. Ball Drop and House Edge reuse SlamWave (point-spawn the ring at the predicted tile). Chip Rake = a row of Area2D pickups in the 'enemies' group. NEW small Wager system: a lightweight banner (existing UI), a 1.5s sector-vote window reading the player's current wedge, a randf() resolve, and a temporary stat modifier applied to the player's stat block for the fight (reuse the relic/stat buff plumbing). The auto-fire-disable is a 6s flag on the gun's can_fire check.

### Mephistopheles, the Quiet Investor — a charming devil that never attacks — it just keeps offering you irresistible deals, and the floor is the small print
- **Spectacle:** A slender, impeccably dressed devil seated calmly in the center of the arena at a tiny desk, legs crossed, sipping something, never moving. No projectiles, no slams — instead, beautiful golden CONTRACT CARDS drift gently across the screen with glowing red sigils, and the arena floor slowly fills with elegant cursive 'fine print' that ignites where you stand too long. Pixel art reads as a serene devil, floating parchment cards with big YES/NO sigils, and creeping red text-fire on the floor.
- **Signature mechanic:** THE CONTRACT. Mephistopheles does not deal damage directly. Instead it periodically presents a deal card with a clear up-front BOON and a clearly-listed CLAUSE: e.g. 'Take +50% damage now. Clause: your dash is sealed.' or 'Heal to full. Clause: auto-aim only targets the FARTHEST enemy.' or 'Triple XP this fight. Clause: you can no longer shoot while moving OR standing — fire only triggers in the 1s after a dash.' You ACCEPT by walking onto the card's glowing sigil, DECLINE by ignoring it until it fades. Every accepted clause permanently (for the fight) alters YOUR combat model. The boss's only kill condition is the Debt Meter: each active clause feeds Debt, and when Debt is high the floor fine-print spreads fast enough to corner you. You damage the boss normally — but the deals you take to do it more handicap how you can fight.
- **Attack patterns (note: all 'attacks' are environmental/contractual, since it never strikes):**
  1. Fine Print Creep (telegraph -> execution -> dodge): cursive red text slowly written across floor tiles, glowing brighter for ~1.0s before each line 'signs' (ignites) -> standing on signed text burns; -> keep walking to outrun the spread, which means you're moving and NOT shooting. The more clauses you hold, the faster it writes.
  2. The Offer (contract card): a card drifts in with sigil + boon + clause text -> you have ~3s to walk onto it (accept) or let it pass (decline) -> accepting instantly applies the boon and stamps the clause. No dodge — this is a decision, and indecision (standing on the sigil by accident while dodging fire) can sign you up unintentionally.
  3. Margin Call (high Debt only): if your Debt Meter maxes, the devil finally snaps its fingers — the entire floor's accumulated fine print ignites at once after a big screen-edge-to-center sweep telegraph (~1.2s) -> dash to the one clean sigil-circle around the desk -> survive the sweep. The only time it 'attacks,' and it's your own unpaid contracts collecting.
- **Exploits the combat model:** This is the only boss whose clauses directly EDIT the no-aim/shoot-while-still/dash rules as gameplay. Clauses can seal your dash (now you can't counter telegraphs), invert auto-aim to farthest (your fire wanders off the boss onto trash), or break the still-to-shoot contract itself (e.g. fire only in a window after dashing, forcing a whole new rhythm). You take these handicaps voluntarily because the boons are huge — so the player knowingly degrades their own core mechanics for power. And Fine Print Creep is the ultimate shoot-while-still trap: the floor is lava that grows the more deals you've signed, so the greedier you got, the less you're allowed to stand still and shoot.
- **Phases:** It escalates by your CHOICES, not just its HP. With 0-1 clauses: gentle fine-print creep, generous-looking offers. At 2+ clauses (or 50% HP): offers get richer but clauses get nastier (now touching the combat model directly), Debt rises faster. Near death OR at max Debt: Margin Call. A no-deal pacifist run keeps the boss nearly harmless but slow to kill; a max-greed run is a frantic self-sabotaged bullet-floor.
- **Counterplay:** The skill is contract literacy — read the clause before stepping on the sigil, take only deals your build can absorb, and decline anything that seals dash if you can't kill fast. Spatial discipline matters because accept = walking onto a sigil, so you must dodge fine-print without accidentally signing. The lesson: in a greed boss the most dangerous damage is the power you accepted, and 'no' is a valid, sometimes optimal, build choice.
- **Difficulty knobs + wave depth:** Tune offer frequency, boon strength vs clause severity, Debt-per-clause rate, fine-print spread speed, and the Margin Call telegraph time. Debut at wave 25 (it needs a player who deeply understands their own combat model to subvert it meaningfully). Recurs at 35+ where clauses stack without a cap.
- **Reward twist:** Payout scales with clauses-held-at-death (greed): die-with-many = a guaranteed rare/legendary relic, but a small chance the relic is itself 'cursed' (a relic with its own clause that carries into the next run as a relic+drawback). A pure no-deal clear drops a clean 'Voided Contract' relic that grants a one-time decline-and-keep on a future Mephistopheles offer. All clauses vanish when the boss dies.
- **Godot build notes:** No SlamWave needed except for the Margin Call sweep (one big point/edge ring). Fine Print Creep = a tile/grid of Area2D damage zones revealed progressively with a _draw() text/scribble overlay and a hit-flash shader on ignition. NEW Contract system: offer cards are simple UI/CanvasItem nodes with an Area2D 'sigil' the player overlaps to accept and a Timer to expire (decline); on accept, apply boon via the existing relic/stat buff plumbing and set a clause flag that the player's gun/dash/auto-aim scripts check (e.g. can_dash, aim_mode = FARTHEST, fire_window). Debt Meter is a single float (like GreedMeter) driving fine-print spread rate and the Margin Call trigger. The clause flags are the only genuinely new wiring — small boolean/enum hooks added to the existing player controller and gun.

### The Piñatasaur — a swollen papier-mâché beast that rains loot the longer it lives — and gets deadlier with every coin it bleeds
- **Spectacle:** A garish neon-striped donkey-dragon the size of the screen, stitched from crepe paper, visibly bulging and jiggling. Every few seconds it splits a seam and a fountain of coins, candy, and XP gems sprays out in an arc — a slot-machine waterfall on a phone screen. As it takes damage, cracks of golden light leak from its hide; by the end it's a glowing, overstuffed balloon about to pop. Pixel art reads instantly: bright primary-color stripes, a fat silhouette, and the unmistakable confetti-burst of loot.
- **Signature mechanic:** GREED METER. A visible meter fills the whole time the boss is alive. Higher meter = more and rarer loot dripping out per second (coins -> XP gems -> a guaranteed relic at max) BUT every attack scales its size, speed, and frequency off the same meter. Killing it FAST = small clean reward; letting it live = screen-filling jackpot you may not survive. The player literally chooses their own difficulty/reward curve in real time.
- **Attack patterns:**
  1. Confetti Spray (telegraph -> execution -> dodge): the seam it's about to burst glows white and the body inflates for ~0.8s -> a wide cone of fast candy projectiles erupts from that seam, the cone angle widening with the Greed Meter -> dash sideways out of the cone arc; standing in it to keep shooting is the greedy gamble.
  2. Coin Tantrum (SlamWave reuse): it stomps, faint filled-circle telegraph under itself -> expanding ring slam, ~1.0s warning -> dash through the ring band. At high meter it chains two rings back-to-back so you must dash, land, then dash again.
  3. Loot Lure (auto-aim trap): periodically it flings a fat glittering 'loot decoy' gem that lands near the player and sits there tagged as an enemy. While it exists, your auto-aim locks onto the shiny decoy instead of the boss, wasting your DPS — and the decoy explodes into a candy nova after 1.5s (clear pulsing red ring telegraph) if not destroyed or fled.
- **Exploits the combat model:** This is the purest greed-vs-DPS lever. Loot drips faster the longer it lives, so you WANT to stand still and shoot for max payout — but standing still is exactly where its widening Confetti cones and chained Coin Tantrum rings punish you. And the Loot Lure decoy spawns NEXT to you, so auto-aim-nearest yanks your fire off the boss onto a worthless gem the instant you try to farm it. Every second of greed is a second your auto-fire might be pointed at the wrong target while a cone fills your tile.
- **Phases:** Phase shifts are driven by the Greed Meter crossing thresholds rather than only HP. At meter 33%: single cones + single rings, gentle drip. At 66%: cones widen, rings chain x2, decoys spawn. At max meter (or below 25% HP, whichever first): it goes 'sugar rush' — strobing rainbow, constant drip of XP gems, cones fire in a rotating fan, and a relic is guaranteed on death. If you cap the meter you're in a bullet-hell jackpot room.
- **Counterplay:** A skilled thumb decides a target meter, farms to it, then commits to the kill — dashing cones, weaving rings, and shooting the boss in the gaps while ignoring decoys (or kiting them away so auto-aim re-locks the boss). The lesson: reward is a resource you spend HP/risk to buy, and you must pick your exit before the meter outruns your survivability.
- **Difficulty knobs + wave depth:** Tune meter fill rate, loot-per-tick curve, cone-width scaling, ring chain count, and decoy spawn cadence. Debut at wave 10 (early enough to teach the risk/reward gamble, late enough that players have a build to gamble with). Reappears at 25/35 with a faster baseline meter.
- **Reward twist:** Reward scales continuously with how long it lived (Greed Meter at the moment of death), not a fixed table. Pop it instantly = a handful of coins; ride it to max = a relic plus a screen-clear of XP gems that level you up multiple times at once. Optionally: dying to it while the meter is maxed still drops the loot, a darkly funny consolation.
- **Godot build notes:** Reuse SlamWave for Coin Tantrum (just parameterize ring count off the meter). Confetti cones = a simple bullet emitter firing in an arc, angle from meter. Loot Lure decoy = an Area2D in the 'enemies' group (so auto-aim grabs it) plus a _draw() pulsing red telegraph and a Timer to its nova. NEW small system: GreedMeter node holding a float that a Tween/Timer ramps, exposing greed_fraction() the same way Boss exposes health_fraction(); the loot emitter and attack scripts read it. Drip uses your existing XP/coin pickup scenes instanced over time.

### The Pawnbroker — a vault-bellied collector that reaches in and STEALS one of your relics, then wears it as armor you have to repossess
- **Spectacle:** A hunched merchant-creature with a body like a walk-in safe, dozens of tiny grabby gold hands sprouting from its torso, a monocle, and a belly-door that swings open to reveal a glowing hoard. The instant it steals your relic, the relic's icon flies off your HUD and embeds itself glowing on the boss's chest as a literal jewel-plate of armor — you can SEE your stolen power bolted to the thing killing you. On a phone this reads crisp: HUD slot goes dark, a bright relic gem appears pinned to the boss silhouette. Repossessing it shatters the plate in a shower of pawn-tickets.
- **Signature mechanic:** RELIC REPOSSESSION. Early in the fight the Pawnbroker grabs one of your equipped relics (telegraphed steal you can dash to interrupt). The relic's buff is removed from you AND its effect is granted TO the boss (steals a damage relic -> it hits harder; steals a fire-rate relic -> it attacks faster; steals a regen relic -> it heals). The stolen relic becomes a separate destructible armor node on its chest with its own small HP bar. Break that node to instantly snatch your relic back mid-fight, removing the boss's buff and restoring yours. You're fighting a version of yourself, powered by your own gear, and you must repo it to win cleanly.
- **Attack patterns:**
  1. The Grab (steal telegraph -> execution -> dash interrupt): one of its long gold hands extends slowly toward the player, a bright dotted tether line connecting hand-to-player for ~1.2s -> if it reaches you it yanks a relic; -> DASH to snap the tether before contact. Dodging the grab entirely means it never gets your relic (and never gets buffed) — a high-skill clean run.
  2. Repo Pulse (SlamWave reuse): it slams its vault-door shut, faint circle telegraph -> expanding ring -> dash through. Faster if it stole a fire-rate relic.
  3. Coin Shotgun (telegraph -> execution): the belly-door cracks open and fills with light for ~0.9s -> a tight spread of heavy coins blasts forward in the player's direction -> dash perpendicular. Hits harder if it stole a damage relic.
  4. Decoy Hoard (auto-aim trap): it spits out 2-3 fake glowing 'relic' gems that scatter and sit as enemy-tagged dummies. Auto-aim prioritizes the nearest, so your fire splits onto worthless decoys instead of the real chest-plate you need to break.
- **Exploits the combat model:** It turns your own progression against the no-aim model. Because auto-aim targets NEAREST, the stolen-relic armor node and the decoy hoards compete with the boss body for your bullets — to repo your relic you must let the chest-plate become the nearest target (position so auto-aim picks it) while dodging, which conflicts with the still-to-shoot rule. The Grab tether is a movement check: dashing to break it costs you a still-frame of DPS, but ignoring it costs you a relic AND arms the boss. Every defensive choice trades your build's power.
- **Phases:** 100-66%: it attempts The Grab; if successful it equips your relic and the chest-plate node appears. 66-33%: if you haven't repo'd the relic, it tries to steal a SECOND one (now two plates, two buffs); decoy hoards begin. Below 33%: 'liquidation' — it cracks open fully, exposing the hoard, and if it still holds your relics it attacks with their full stacked power, but the plates take double damage so a focused player can repo both in a frantic finish.
- **Counterplay:** Either dash-break every Grab tether for a flawless no-theft kill, or accept the theft and prioritize breaking the chest-plate to repo your relic and de-buff the boss. Learn to walk so the plate becomes auto-aim's nearest target, and to ignore decoys. The lesson: your relics are leverage in both directions — losing one isn't permanent, it's a sub-objective to reclaim.
- **Difficulty knobs + wave depth:** Tune grab tether speed/reach, chest-plate node HP, how many relics it can steal, decoy count, and the liquidation damage multiplier. Debut at wave 15 (player must own at least one relic for the mechanic to fire; if they have none, it falls back to stealing a temporary stat). Encore at 30 stealing two relics off the bat.
- **Reward twist:** Any relic it still holds at death drops back to you guaranteed, PLUS it coughs up one extra relic from its hoard as 'interest.' If you completed the fight without ever letting it steal anything (no-theft clear), it drops a rare relic as a perfect-repo bonus.
- **Godot build notes:** Reuse SlamWave for Repo Pulse; Coin Shotgun is a directional spread emitter; decoys are enemy-group Area2D dummies (same as other bosses' lure tech). NEW small Relic Theft system: on grab-success, call into the existing relic manager to remove the relic's modifier from the player and apply the equivalent buff to the boss, then spawn a child Area2D 'plate' node (in 'enemies' + 'boss-armor' groups) with its own HP that, on death, reverses the swap. The Grab tether is a Line2D + _draw() with a Timer; dash-overlap during the window cancels it.

### Choirmother — The Hymn That Heals
- **Spectacle:** A grotesque pipe-organ fused to a swollen mother figure, her body a cluster of brass pipes wheezing colored smoke. Three fat glowing LARVAE crawl around her, each connected to her by a pulsing musical-staff beam (visible sine-wave line in a distinct color — cyan, magenta, lime). When a larva sings, music-note particles flow UP the beam into her and her HP bar visibly ticks RIGHT. On a phone the three colored heartbeat-lines radiating from a central blob are unmistakable.
- **Signature mechanic:** **Heal-from-adds invulnerability gating.** The Choirmother takes only TOKEN damage (or none) while any larva is alive, and each living larva heals her for a chunk per second via its beam. You must kill all three larvae to make her vulnerable — but the larvae respawn over time and are the NEAREST targets, so your auto-aim is constantly pulled OFF the boss and onto them. It's an attrition race: out-DPS her self-heal by clearing adds faster than they respawn, then burst the brief vulnerable window.
- **Attack patterns:**
  - *Lullaby (heal):* telegraph -> the three beams brighten and pulse in rhythm -> notes flow up, boss heals; if all three sing together a 'chord' flashes and the heal doubles for 2s -> counter: kill larvae to cut beams; a severed beam goes dark with a discordant screech.
  - *Crescendo Ring (reuse SlamWave, sonic):* telegraph -> faint filled ring + a visible audio-ripple -> expanding ring, leading band -> dash through the gap.
  - *Pipe Blast:* telegraph -> one brass pipe glows and points a thin bright line at the player for 0.7s -> a fast straight beam fires down that line -> dash perpendicular. Aimed at wherever you're STANDING (i.e., where you'd be DPSing), so it punishes the stationary burst window.
- **Exploits the combat model:** This is the auto-aim-nearest exploit in its purest form. The larvae are deliberately placed closer than the boss, so the gun REFUSES to shoot the boss while any add lives — the player can't override aim. Worse, killing larvae requires standing still (shoot-only-while-still), but standing still parks you in Pipe Blast / Crescendo range. The core tension: every second you spend shooting larvae from a safe spot, she heals; every second you dodge, you're not clearing adds AND not damaging her. The player has to find stand-still windows long enough to clear all three before they respawn.
- **Phases:** 100-66%: 3 larvae, slow respawn, per-larva heal modest. 66%: larvae respawn faster and gain a small shield (2 hits to pop), heal-per-larva up. 33%: 'Final Chorus' — she summons a FOURTH larva and the beams turn into damaging lasers (standing in a beam now hurts), so you must clear adds while also dodging the very tethers you're trying to sever.
- **Counterplay:** Skilled play = prioritize the larvae as a clear order, use a high-fire-rate or piercing weapon to chew them, and bank dashes for Pipe Blast specifically. The lesson: in an auto-aim game, sometimes the enemy you WANT to kill isn't the one the gun will shoot — you must manipulate proximity (back away so the boss isn't 'nearest' relative to a clump, or move so a larva is closest) to steer your own fire.
- **Difficulty knobs + wave depth:** Heal-per-larva, larva respawn timer, larva HP, vulnerable-window length, beam-laser damage in P3. Debut wave 15 (needs the player to have multishot/pierce options). At 25+ enable larva shields and the laser beams.
- **Reward twist:** Killing a larva drops a small 'silenced note' XP shard, so add-clearing still feeds level-ups — and her death relic is 'Discord' (small chance your bullets disable enemy healing on hit).
- **Godot build notes:** Extend Boss.gd; override take_damage to ignore/reduce damage while `living_larvae > 0`. Larvae = Enemy.gd instances in the 'enemies' group (so auto-aim grabs them), spawned as children, each holding a ref back to the boss; in _process each living larva calls boss.heal(rate*delta). Beams = a _draw sine-wave line from boss to each larva, brightening on heal tick. Respawn = a Timer per slot. Reuse SlamWave for Crescendo. NEW small systems = the heal-link tick + the optional beam-becomes-laser hurtbox (an Area2D along the beam in P3).

### Null Verse — The Roaming Hush
- **Spectacle:** A black obelisk wrapped in floating inverted text, hovering above the arena, utterly still while it sends out slow-drifting BUBBLES of grayscale void. Inside a hush bubble the world desaturates to black-and-white, the music cuts to a dead hum, and the player's muzzle flashes sputter out. On a phone the bubbles read as soft-edged gray circles sliding across the screen, and the desaturation when you enter one is an unmistakable full-screen 'something is WRONG' cue.
- **Signature mechanic:** **Disarming roaming invuln zones (inverted).** The boss is invulnerable EVERYWHERE EXCEPT inside its own hush bubbles — but a hush bubble JAMS your gun (forces hold_fire) and LOCKS your dash. So the only place you can damage it is the one place you can't shoot or dodge. The trick: the boss itself periodically drifts INTO a hush bubble; for the brief overlap you can stand in that same bubble and... still can't fire. The real solution is the phase mechanic below (a relic/ammo charm that fires once on entry, or building a 'hush-pierce' window). Until then it's a roaming-zone positioning nightmare where safety and lethality are inverted.
- **Attack patterns:**
  - *Hush Drift (signature zone):* telegraph -> a gray bubble blooms at the obelisk (0.8s fade-in) then drifts slowly across the arena -> standing inside sets hold_fire=true and disables dash; the boss is only damageable when both it and the player are inside the SAME bubble -> counter: read bubble paths, surf the edges, time entries.
  - *Mute Pulse (reuse SlamWave, inverted color):* telegraph -> faint gray filled ring -> expanding WHITE-on-black ring, leading band -> dash out — but if you're standing in a hush bubble your dash is locked, so you must EXIT the bubble first then dash, a 2-step reaction. Teaches you to never get cornered in a hush.
  - *Recite:* telegraph -> three lines of glowing inverted text orbit the obelisk and lock onto the player's current tile for 1.0s -> three slow homing glyph-bolts launch -> dash to break the lock (only works if not hushed). Punishes standing still, classic.
- **Exploits the combat model:** This is the most direct assault on the combat model: it literally TOGGLES the two systems the player depends on. hold_fire is the exact flag the Player already sets while moving — the hush forces it true even while standing still, so 'stand still to shoot' stops working. And dash-lock removes the universal counter to every telegraph. The boss weaponizes the player's own engine flags against them. The fight becomes about managing WHERE the disarm zones are vs where you must stand to deal damage, with auto-aim still trying to fire (and being silenced) so you get constant feedback on the jam.
- **Phases:** 100-66%: one slow bubble at a time, generous gaps, dash-lock only (you can still walk out). 66%: two bubbles overlap-drift, Mute Pulses come during hushes (forcing the exit-then-dash combo). 33%: 'Total Hush' — the obelisk floods most of the arena with bubbles leaving thin clear lanes; you ride the lanes, dipping into the boss's bubble only in the split-second it overlaps a clear edge to land your shots.
- **Counterplay:** Skilled play = treat hush bubbles like lava you sometimes must touch; pre-aim by positioning so that when you DO get a firing window (boss + you sharing the bubble's edge where the jam flickers, or via a hush-pierce relic) your auto-aim is already pointed right. Bank dashes and never enter a bubble without a clear exit. The player learns that their auto-systems are not guaranteed — and to play around losing them.
- **Difficulty knobs + wave depth:** Bubble count/size/drift speed, whether a hush fully jams or just halves fire rate (early version = fire-rate penalty instead of full jam, much gentler), dash-lock vs dash-shortened, P3 lane width. Debut wave 20 (this is a late, brain-bending fight; players need relic options like a 'hush-pierce' charm available by then). At 30+ go full jam + Total Hush.
- **Reward twist:** Drops the relic 'Last Word' — once per hush entry your gun fires a single free shot the instant you enter a silence zone, turning its own disarm into your only damage window (and hinting at the intended solution). A gorgeous mechanical payoff.
- **Godot build notes:** Extend Boss.gd; override take_damage to ignore damage unless `_player_in_my_field`. HushField = Area2D bubbles (CollisionShape2D circle) that lerp across the arena; on player body_entered set a flag the Player reads to force gun.hold_fire=true and block dash (add a `disarmed` bool the Player/Gun already-existing hold_fire + dash gate respects). Desaturation = a small canvas_item/screen shader toggled while player is hushed (NEW small screen-space shader — flag it, keep it to one grayscale param). Reuse SlamWave (recolored) for Mute Pulse and Recite bolts can be simple homing Bullet.gd instances in a hostile group. NEW small systems = the roaming HushField zones + the grayscale screen shader; both are intentionally minimal.

### The Genuflect — Mirror Saint
- **Spectacle:** A huge, cracked alabaster statue of a kneeling saint, hands pressed in prayer, a halo of slowly-rotating golden glyphs. On a phone it reads instantly: a bright golden ARC fan (its prayer cone) sweeps in front of its face, and a glowing CRACK runs down its exposed back. When it reflects, your own bullet tracers flip from white to gold and arc back at you — an unmistakable "oh no, those are MINE" moment.
- **Signature mechanic:** **Front-facing reflection parry.** The saint has a ~120-degree golden prayer cone on its FRONT. Any bullet that enters the cone is not absorbed — it is *reflected back toward the player as a hostile projectile* and the boss takes ZERO damage. Its back (a glowing spine-crack) is its only damageable hitbox. Because auto-aim always fires at the boss's center from wherever the player stands, the player must physically circle to the rear to land any damage at all — a pure positioning puzzle in a no-aim game.
- **Attack patterns:**
  - *Reflect (passive stance):* telegraph -> the golden cone is always drawn, brightening for 0.4s before it actively reflects -> any bullet in-cone returns as a gold tracer dealing your-own-damage back to you -> dodge by NOT shooting from the front; reposition behind. Dash is for repositioning, not for dodging the reflect itself.
  - *Slow Turn:* telegraph -> the saint's head and cone rotate to track the player at a deliberate ~45 deg/sec (visible sweeping arc) -> if the player is in the cone when it 'locks' (cone flares solid for 0.3s) the next reflect is doubled -> counter: keep orbiting faster than it can turn so its back stays exposed.
  - *Penance Slam (reuse SlamWave):* telegraph -> faint filled ring at its base -> expanding gold ring, leading band hits -> dash out. Fires only when the player is directly behind it, punishing the very spot you must stand to deal damage.
- **Exploits the combat model:** Auto-aim ALWAYS targets nearest-point-of-boss and you can't manually aim, so a frontal reflect means your auto-fire literally weaponizes itself against you the instant you stop to shoot from the wrong side. The fix — get behind it — collides with shoot-only-while-still: orbiting to the rear means moving (no DPS), and the moment you stop to fire from behind, the Penance Slam punishes that exact tile. The boss turns 'stand still to deal damage' and 'move to position' into a constant tug-of-war.
- **Phases:** 100-66%: cone is 120deg, turns slowly. 66%: cone widens to 160deg (smaller safe rear arc) and adds a second reflect color (red tracers home slightly). 33%: the saint 'shatters its kneeling pose' and begins slowly walking while still facing-locked — now you must orbit a moving target whose back arc keeps sliding away; Penance Slams come in pairs.
- **Counterplay:** Skilled play = treat it like a clock. Orbit counter to its turn, stop only in the rear arc for short bursts, dash out of each Penance Slam, then re-stop. The player learns to read facing cones and to STOP FIRING from the front (the hardest lesson in an auto-fire game — sometimes you want your gun silent).
- **Difficulty knobs + wave depth:** Turn speed, cone angle, reflected-bullet damage multiplier (start at 0.5x your damage so early reflects sting but don't one-shot), slam cadence. Debut wave 10. At 20+ widen the cone and add the homing red reflect.
- **Reward twist:** On death it drops a relic 'Saint's Spine' — your bullets gain a small backstab bonus vs all future bosses (thematic payoff for learning the rear-attack lesson).
- **Godot build notes:** Extend Boss.gd. Add a `facing` Vector2 it lerps toward the player slowly. Reflection = an Area2D 'cone' child (a thin pie CollisionPolygon2D drawn via _draw as a gold arc); on bullet body_entered while bullet is inside the cone angle, call bullet.reflect() — reuse Bullet.gd by zeroing its damage-to-enemy and re-pointing velocity at the player + adding it to a 'hostile' group the Player's hurtbox reads. Only the rear Area2D (the spine-crack hitbox) routes hits to take_damage(). Reuse SlamWave verbatim for Penance Slam, gated on player-is-behind. NEW small system = the cone reflect; everything else is existing primitives.

### Aegis Tortoise — The Wandering Bastion
- **Spectacle:** A mountainous spiked-shell tortoise that lumbers across the arena leaving cracked-tile footprints. When it turtles, the shell SLAMS shut with a metallic clang, plates ringing with a blue ricochet sheen — your bullets visibly *spark and bounce off* (deflection sparks via particles), and a big 'CLINK' damage-number-zero floats up. When it overextends into a body-slam and flips, its pink soft underbelly is exposed and pulses — a screaming 'SHOOT HERE' beacon. The flip-and-belly-up is the screenshot moment.
- **Signature mechanic:** **Bait-the-overextend underbelly window.** The tortoise is permanently armored (near-immune, bullets deflect) EXCEPT in the recovery window after it commits to a lunging Body Slam, when it rolls belly-up and exposes a soft glowing underbelly hitbox for ~2.5s. You can't force this — you must BAIT it: stand at range so it commits to a slam, dodge the slam, then dump DPS into the belly before it rights itself. It's a pure attrition/timing puzzle: no bait = no damage, ever.
- **Attack patterns:**
  - *Turtle (deflect stance):* telegraph -> shell plates glow blue and snap shut -> all incoming bullets deflect (0 dmg, spark particles) for its duration -> counter: stop wasting bullets, reposition to bait a slam.
  - *Body Slam (the bait, reuse SlamWave):* telegraph -> it rears up, a long faint LUNGE rectangle telegraph points at the player for ~1.0s -> it hurls forward along that line (a directional SlamWave band) and OVERSHOOTS, flipping belly-up at the end -> dash sideways out of the lunge lane; then the belly is open.
  - *Spin Cyclone:* telegraph -> shell spikes flare, a growing ring telegraph -> it spins in place flinging an expanding spiked ring (SlamWave reskin) -> dash through; this attack does NOT expose the belly (it's a 'denied window' to teach patience).

  - 'FERMATA' (hold-the-note freeze): telegraph -> a baton freezes high + a held-note symbol pulses for ~1.0s -> execution: all REAL enemies freeze in place for ~2s (great for you) BUT the conductor fires a slow expanding ring meanwhile -> dodge: use the freeze to reposition for the next record AND dash the ring.
  - 'CRESCENDO' (overlap beam): telegraph -> all four batons point at you, four faint converging lines draw over ~1.1s -> execution: four SlamWave bands sweep inward to your position simultaneously -> dodge: dash to the one safe seam between bands (there's always a readable gap).
  - 'GRAND FINALE' (phase 3): telegraph -> the staff floods gold, two recordings start STACKING -> execution: two ghost-armies + two ghost-yous replay at once, layering your last 6 seconds -> dodge: requires clean, deliberate footwork during the record so both your ghosts are parked somewhere safe.
- **Exploits the combat model:** It attacks shoot-only-while-still from a brand-new angle: your standing-still firing spots get recorded and then a damaging ghost-you re-occupies them — so your safest, highest-DPS habit (plant and shoot) literally becomes where the danger spawns next cycle. It also doubles the enemy count for free via auto-aim-nearest: half your bullets chase phantoms. And it makes DASH double-purpose — not just dodging attacks but dodging your own past self. The deepest insight: the player must think about where they were, not just where they are.
- **Phases:**
  - 100-66%: single 3s record/playback loop, ghost-you only chips, no CRESCENDO. Teaches 'my path becomes a hazard'.
  - 66-33%: faster loop, ghost-you full contact damage, FERMATA + CRESCENDO added. Now footwork during recording matters.
  - 33-0%: GRAND FINALE stacking double-echoes; the staff scrolls faster; recordings start with almost no sepia warning so you must assume you're always being recorded. Two translucent Ryans on a crowded screen = peak spectacle.
- **Counterplay:** Treat every moment as a recording: take clean, economical paths and do your standing-DPS in spots you can vacate, so your future ghost parks harmlessly. Dash THROUGH ghost-you on overlap. Ignore blue targets — pre-aim by standing near real zombies. Learns rhythm and spatial foresight: 'play a line I'd be happy to hear played back.'
- **Difficulty knobs + wave depth:** Record length, playback speed, ghost contact damage, loop gap, ghost-aim weighting (could make ghosts un-targetable on easy), CRESCENDO band count, FINALE stack count. Debut wave 30+. Wave-25 'rehearsal' variant: ghosts visible-but-untargetable and non-damaging, pure spectacle teaching tool, only FERMATA active.
- **Godot build notes:** The only real new system is EchoRecorder.gd: each _physics_process push {pos, facing} for player + each live enemy + boss into per-entity Arrays (cap length = record_frames). EchoPlayback ghosts are plain Sprite2D+Area2D nodes (modulate blue, alpha ~0.5) that, each frame, set global_position from the buffer index; add them to 'enemies' so existing auto-aim picks them up (set their take_damage to no-op so bullets pass/expire). Ghost-you uses the player sprite tinted blue and calls player.take_damage on body overlap (rate-limited). Reuse SlamWave for FERMATA ring + CRESCENDO bands (four band-variant instances). Reuse flash.gdshader. Keep buffers fixed-size to bound memory; sample at physics tick, not every frame, to keep it cheap on phones.


## Family: Environmental Set-Piece (Living Arena) (1)

### Atlas Unbound, the Arena That Stood Up — The fight has no boss on the field; the ARENA itself is the boss: the walls rip free, fold into a colossal titan built from the floor you're standing on, and you fight it on the shrinking scraps of ground it hasn't yet pulled into its own body.
- **Spectacle:** Boss music hits and... nothing spawns. Then the arena GRINDS. The border walls crack, peel up off the screen edges, and assemble at the top into a vast humanoid titan whose body is visibly made of arena tiles — you can see the patch of floor it's wearing as a chestplate. A single molten HEART glows in the gap between its tiles. As the fight goes the titan reaches DOWN and rips more tiles out of the floor you're standing on (those squares go dark/void), flying them up to armor itself — so the literal playable ground shrinks and rearranges. The screenshot: a giant made of your own arena looming over a cracked, half-missing floor with you marooned on a glowing island of remaining tiles. Pixel art sells this — detached tiles tumble with the same texture as the floor.
- **Signature mechanic:** THE FLOOR IS THE BOSS'S HP. The titan is invulnerable everywhere except its exposed HEART, which is only reachable/hittable when it leans down to harvest tiles. Meanwhile it strip-mines the arena: tiles you're not standing on get ripped up into void gaps (instant-fall hazard if you're caught moving over one as it lifts). To DAMAGE it you must bait it into a harvest-lean (it always reaches toward the largest remaining tile-cluster), then shoot the dipped Heart during the lean window — but standing on a soon-to-be-harvested cluster to bait it means standing where the floor is about to vanish.
- **Attack patterns:**
  - 'STRIP-MINE' (the core loop / weak-point window): telegraph -> 4-6 floor tiles flash white with a rising hum for ~1.0s (marked for harvest) -> execution: those tiles lift and fly up; the titan LEANS down to grab them, dipping its Heart into firing range for ~2.5s -> dodge/counter: dash OFF the flashing tiles before they lift (falling = a big hit / brief stun), then plant on solid ground and unload into the exposed Heart during the lean.
  - 'WALL SLAM' (returns a limb to the floor): telegraph -> a titan arm rears back, a wide rectangular shadow falls on a strip of floor over ~1.1s -> execution: the arm slams that strip (a SlamWave-style band along a rectangle), and slammed tiles are knocked loose into gaps -> dodge: dash out of the shadow strip; note it permanently shrinks your floor.
  - 'TILE HAIL' (ranged): telegraph -> the titan flings 5-8 loose tiles; each casts a small shrinking shadow-marker where it'll land over ~0.9s -> execution: tiles crash down as one-hit impacts, then become temporary cover/obstacles -> dodge: weave through the marked landing spots; the landed tiles can REFILL a void gap (rare bit of floor you get back).
  - 'TECTONIC TILT' (phase 3): telegraph -> the whole remaining floor cluster glows and slowly cants -> execution: everything (you + loose tiles) slides toward the void edge while the titan does STRIP-MINE -> dodge: fight the slide by moving uphill — which means moving = no DPS during the one window you most need it.
- **Exploits the combat model:** Two layers. First, it makes STANDING STILL — your only firing state — terrifying, because the ground under a still player is exactly what the titan harvests; you must constantly re-earn a stable firing platform. Second, the only damage window (the Heart lean) requires you to first bait the harvest by occupying the doomed cluster, then dash off and re-plant in time to shoot — a tight move/stop/shoot/move loop that directly weaponizes the move-shoot exclusivity. There are no adds to steal auto-aim because the boss removed the floor instead — a totally different pressure than every swarm boss.
- **Phases:**
  - 100-66%: arena assembles; gentle STRIP-MINE every ~6s, generous ~3s Heart windows, ~70% of floor intact. Teaches 'flashing tiles = leave, then shoot the Heart'.
  - 66-33%: WALL SLAM + TILE HAIL added; harvests come faster; floor down to ~half, fragmented into 2-3 islands you must dash-gap between. Heart window ~2s.
  - 33-0%: 'COLLAPSE' — TECTONIC TILT begins, harvests are near-continuous, floor reduced to a few drifting platforms; the Heart stays exposed longer (it's overextended/desperate) so the DPS check is real but the footing is brutal. Final image: you on one glowing tile, a void all around, a titan of stolen floor folding over you.
- **Counterplay:** Read the white tile-flash as your bait-and-fire cue: stand on a doomed cluster just long enough to commit the harvest, dash to your pre-chosen safe island, plant, and burst the Heart for the lean duration. Bank TILE HAIL drops to repair routes between islands. Learns deliberate platform management and the move/stop rhythm under spatial loss — the opposite of stand-and-spray.
- **Difficulty knobs + wave depth:** Harvest interval, tiles-per-harvest, Heart window length, floor-return rate from HAIL, slide speed in TILT, min-floor floor (how small it's allowed to get), fall-damage vs. stun on dropping into a gap. Debut wave 30+ as a true set-piece finale. Wave-25 'tremor' variant: smaller titan, no TILT, floor never drops below ~60%, longer Heart windows.
- **Reward twist:** On death the titan COLLAPSES and all its stolen tiles rain back down, instantly restoring the full arena floor (a satisfying 'you got your home back' beat) plus the standard XP burst, full-heal, and relic. Optional relic: 'Bedrock' — first fall into a void gap each run is harmless.
- **Godot build notes:** NEW small system: TileFloor.gd manages a 2D array of tile Sprite2D nodes (the arena floor split into a grid). Harvest = pick a cluster, flash via flash.gdshader, then Tween the chosen tiles' global_position up to the titan and disable their collision (or mark their cells as 'void' in a lookup the Player checks each physics tick: if the player's cell is void and not dashing, apply the fall hit). The Heart is an Area2D weak-point on the titan whose take_damage only applies while a `var leaning := true` flag is set (toggled by the STRIP-MINE Tween). Reuse SlamWave for WALL SLAM (rectangular band variant) and TILE HAIL (each tile = a shrinking-shadow marker then an impact band). TECTONIC TILT = add a constant velocity nudge toward the void edge in Player._physics_process while a boss flag is set (and apply the same drift to loose tile nodes). No boss CharacterBody2D chase needed — the titan is static at the top edge; put the Heart in the 'enemies'/'boss' groups so the HUD bar + auto-aim still work when it's exposed.
